# 知识点归纳流程与题目解析
	1. 把一个PDF文档中的题目截图放入一个markdown格式的题库中
	2. 把一个PDF文档的答案截图放入题库中 ，放置在题目下面 
	3. 把答案和题目放入AI studio得到完整解析 加 衍生考点 ，把这个解析复制粘贴放入markdown格式的题目解析库 
	4. 把题目解析分解，提取重点 ，完善知识点 ，放入题库
	- 参考prompt
	   这些题目的答案怎么推出来的，请你基于这些题目说提及的知识点，详细介绍一下，并介绍可能衍生出的考点，你回答的数学公式用LaTeX表示，$...$格式的，不要有空格和‘’在$...$ 的附近
