#积分收敛 
#含有多个瑕点的反常积分收敛的充要条件 
1.   第一部分：分析 $\int_{0}^{1} \frac{1}{x^a(1+x)^b} dx$ 的收敛性
	1. 积分的瑕点（可能导致积分发散的点）是$x=0$ 
		1. #极限比较判别法 来判断其收敛性
			1. 确定被积函数和比较函数
				1. 被积函数是 $f(x) = \frac{1}{x^a(1+x)^b}$。
				    *   当$x \to 0^+$时，$(1+x)$项趋近于$1$，所以$(1+x)^b$趋近于$1^b=1$。此时，$f(x)$的行为主要由$x^a$决定。因此，我们选择一个更简单的函数$g(x) = \frac{1}{x^a}$作为比较函数
				2.  **计算极限**：
				    $\lim_{x \to 0^+} \frac{f(x)}{g(x)} = \lim_{x \to 0^+} \frac{\frac{1}{x^a(1+x)^b}}{\frac{1}{x^a}} = \lim_{x \to 0^+} \frac{1}{(1+x)^b} = \frac{1}{(1+0)^b} = 1$
				    因为极限值$1$是一个不为0的常数，所以$\int_{0}^{1} \frac{1}{x^a(1+x)^b} dx$与$\int_{0}^{1} \frac{1}{x^a} dx$的敛散性相同。
			2.  **判断比较函数的收敛性**：
			    $\int_{0}^{1} \frac{1}{x^a} dx$是p-积分的一种形式。
			    **核心知识点：p-积分在0点的敛散性**
			    反常积分$\int_{0}^{c} \frac{1}{x^p} dx$ (其中$c>0$)，当$p<1$时收敛，当$p \ge 1$时发散。
			    因此，为了使$\int_{0}^{1} \frac{1}{x^a} dx$收敛，必须满足$a<1$。
2. 第二部分：分析 $\int_{1}^{+\infty} \frac{1}{x^a(1+x)^b} dx$ 的收敛性
	1. 这部分积分是无穷区间上的反常积分。我们同样使用 #极限比较判别法 
		1.  **确定被积函数和比较函数**：
	    *   被积函数是 $f(x) = \frac{1}{x^a(1+x)^b}$。
	    *   当$x \to +\infty$时，$(1+x)$中的常数$1$可以忽略不计，所以$(1+x) \sim x$。因此，分母$x^a(1+x)^b \sim x^a \cdot x^b = x^{a+b}$。我们选择比较函数$g(x) = \frac{1}{x^{a+b}}$。
	2. 计算极限
		1. $\lim_{x \to +\infty} \frac{f(x)}{g(x)} = \lim_{x \to +\infty} \frac{\frac{1}{x^a(1+x)^b}}{\frac{1}{x^{a+b}}} =$
			$\lim_{x \to +\infty} \frac{x^{a+b}}{x^a(1+x)^b} = \lim_{x \to +\infty} (\frac{x}{1+x})^b = (\lim_{x \to +\infty} \frac{1}{\frac{1}{x}+1})^b = 1^b = 1$
		2. 因为极限值$1$是一个不为0的常数，所以$\int_{1}^{+\infty} \frac{1}{x^a(1+x)^b} dx$与$\int_{1}^{+\infty} \frac{1}{x^{a+b}} dx$的敛散性相同
	3.  **判断比较函数的收敛性**：
	    $\int_{1}^{+\infty} \frac{1}{x^{a+b}} dx$是p-积分的另一种形式。
	    **核心知识点：p-积分在无穷远点的敛散性**
	    反常积分$\int_{c}^{+\infty} \frac{1}{x^p} dx$ (其中$c>0$)，当$p>1$时收敛，当$p \le 1$时发散。
	    因此，为了使$\int_{1}^{+\infty} \frac{1}{x^{a+b}} dx$收敛，必须满足指数$a+b>1$。
	
	**结论2**：第二部分积分收敛的条件是$a+b>1$。
为了使原积分$I = \int_{0}^{+\infty} \frac{1}{x^a(1+x)^b} dx$收敛，必须同时满足第一部分和第二部分的收敛条件。
即：$a<1$ 且 $a+b>1$
	
- 衍生  
	- [[p-积分]]  不同形式的 #p-积分判别
	    *   $\int_{c}^{+\infty} \frac{1}{x^p} dx$：要求$p>1$收敛。
	    *   $\int_{0}^{c} \frac{1}{x^p} dx$：要求$p<1$收敛。
	    *   $\int_{a}^{b} \frac{1}{(x-a)^p} dx$：瑕点在$a$，要求$p<1$收敛。
	    *   $\int_{a}^{b} \frac{1}{(b-x)^p} dx$：瑕点在$b$，要求$p<1$收敛。
	-   更复杂的被积函数 
		- 例如，判断$\int_{0}^{+\infty} \frac{\ln(1+x)}{x^p} dx$的敛散性。
	        *   在$x \to 0^+$时，$\ln(1+x) \sim x$，所以被积函数 $\sim \frac{x}{x^p} = \frac{1}{x^{p-1}}$。要求$p-1<1$，即$p<2$。
	        *   在$x \to +\infty$时，被积函数与$\frac{\ln x}{x^p}$同敛散。根据p-积分的对数形式，要求$p>1$。
	        *   综合起来，收敛条件是$1<p<2$。
	-  与 #Gamma函数 和 #Beta函数 的关系  [[特殊函数(伽马Gamma函数)]]
		- 本题的积分实际上是**Beta函数**的一种形式。Beta函数的定义之一是 $B(m, n) = \int_{0}^{+\infty} \frac{t^{m-1}}{(1+t)^{m+n}} dt$ 
		-  我们对比题中积分 $\int_{0}^{+\infty} \frac{1}{x^a(1+x)^b} dx$ 和Beta函数的标准形式 $B(p, q) = \int_0^1 t^{p-1}(1-t)^{q-1} dt$。通过变量代换 $x = \frac{t}{1-t}$，可以证明原积分等于 $B(1-a, a+b-1)$
		- #Beta函数收敛域 
			- Beta函数$B(p, q)$收敛的条件是$p>0$且$q>0$ 
			- 因此，我们得到$1-a>0$且$a+b-1>0$。
		    *   这直接给出了$a<1$和$a+b>1$的结论。这是一个更高等但更快捷的方法
	- #绝对收敛和条件收敛 
		对于非正项函数的反常积分，如$\int_{1}^{+\infty} \frac{\sin x}{x^p} dx$，需要讨论绝对收敛 ($\int |\frac{\sin x}{x^p}| dx$收敛)和条件收敛($\int \frac{\sin x}{x^p} dx$收敛但$\int |\frac{\sin x}{x^p}| dx$发散)。
	    *   $\int_{1}^{+\infty} |\frac{\sin x}{x^p}| dx \le \int_{1}^{+\infty} \frac{1}{x^p} dx$。当$p>1$时，$\int_{1}^{+\infty} \frac{1}{x^p} dx$收敛，所以原积分绝对收敛。
	    *   当$0<p \le 1$时，可以证明积分条件收敛（通常使用狄利克雷判别法）。

#乘法法则   #原函数必须连续
- 解法1：不定积分法 + 保证连续性 
1. 第一步：对每一段分别求不定积分 
	1. $F(x) = \int \ln x dx = x\ln x - \int x \cdot \frac{1}{x}dx = x\ln x - x + C_2$
		1. $F(x) = \int 2(x-1)dx = (x-1)^2 + C_1$ 
		2. 得到了原函数$F(x)$的通用表达式：
		$F(x)=\begin{cases} (x-1)^2 + C_1, & x<1 \\ x\ln x - x + C_2, & x\geq 1 \end{cases}$
2. 第二步：利用分段点的连续性确定常数关系
	1. $F(x)$必须在分段点$x=1$处连续。这意味着$F(x)$在$x=1$处的左极限必须等于右极限（也等于函数值）。
		*   计算左极限：
		    $\lim_{x\to 1^-} F(x) = \lim_{x\to 1^-} [(x-1)^2 + C_1] = (1-1)^2 + C_1 = C_1$
		*   计算右极限（即函数值）：
		    $\lim_{x\to 1^+} F(x) = F(1) = 1\cdot\ln 1 - 1 + C_2 = 0 - 1 + C_2 = -1 + C_2$
		令左极限等于右极限：
		$C_1 = -1 + C_2$
	2. 令$C_1=0$。
		根据$C_1 = -1 + C_2$，可得$C_2 = C_1 + 1 = 0 + 1 = 1$
	3. 将$C_1=0, C_2=1$代入$F(x)$的表达式中，得到一个特解：
		$F(x)=\begin{cases} (x-1)^2, & x<1 \\ x\ln x - x + 1, & x\geq 1 \end{cases}$
	4. 将$x\geq 1$的部分进行因式分解，可以写成$x(\ln x - 1) + 1$。
		所以，$F(x)=\begin{cases} (x-1)^2, & x<1 \\ x(\ln x - 1) + 1, & x\geq 1 \end{cases}$
- 解法2： #变上限积分法 
	- 第一步：构造变上限积分函数  
		- 根据牛顿-莱布尼茨公式，函数$F(x) = \int_a^x f(t)dt$是$f(x)$的一个原函数，并且满足$F(a)=0$。这个函数是天然连续的 
		- 计算方便，我们通常选取分段点作为积分下限$a$。这里，我们取$a=1$。
			令$F(x) = \int_1^x f(t)dt$ 
	2. 第二步：分情况计算定积分 
		1. 当$x<1$时，积分上限在积分下限的左边：
	    $F(x) = \int_1^x f(t)dt = \int_1^x 2(t-1)dt = [(t-1)^2]_1^x = (x-1)^2 - (1-1)^2 = (x-1)^2$
		*   当$x\geq 1$时，积分路径上的$t$始终大于等于1：
	    $F(x) = \int_1^x f(t)dt = \int_1^x \ln t dt = [t\ln t - t]_1^x$
	    $= (x\ln x - x) - (1\ln 1 - 1) = x\ln x - x - (0-1) = x\ln x - x + 1$
	1. 第三步：组合成最终结果
	将两段的结果合并，得到：
	$F(x)=\begin{cases} (x-1)^2, & x<1 \\ x\ln x - x + 1, & x\geq 1 \end{cases}$
-  衍生 
	- 原函数的 #可导性 问题
		-  **结论**：如果$f(x)$在某点$x_0$处连续，那么它的原函数$F(x)$在该点一定可导，且$F'(x_0) = f(x_0)$ 
		-   **在本题中**：我们需要判断$f(x)$在$x=1$处是否连续。
	        $\lim_{x\to 1^-} f(x) = \lim_{x\to 1^-} 2(x-1) = 0$
	        $\lim_{x\to 1^+} f(x) = \lim_{x\to 1^+} \ln x = \ln 1 = 0$
	        因为左右极限相等且等于$f(1)=0$，所以$f(x)$在$x=1$处连续。因此，它的原函数$F(x)$在$x=1$处是**可导的**，且$F'(1)=f(1)=0$
	- 原函数的 #二阶可导性 问题
		-  **结论**：原函数$F(x)$的二阶可导性取决于$f(x)$的一阶可导性，即$F''(x) = f'(x)$。
	    *   **在本题中**：我们需要判断$f(x)$在$x=1$处是否可导。
	        *   当$x<1$时，$f'(x) = 2$。所以左导数$f'_-(1)=2$。
	        *   当$x>1$时，$f'(x) = \frac{1}{x}$。所以右导数$f'_+(1)=1$。
	        因为左右导数不相等，所以$f(x)$在$x=1$处**不可导**。
	        *   **衍生结论**：这意味着原函数$F(x)$在$x=1$处是**二阶不可导**的。
	-  **求包含分段点的定积分**
	    *   例如，计算$\int_0^e f(x)dx$。
	    *   需要将积分区间根据分段点拆开：
	        $\int_0^e f(x)dx = \int_0^1 f(x)dx + \int_1^e f(x)dx$
	        $= \int_0^1 2(x-1)dx + \int_1^e \ln x dx$
	        $= [(x-1)^2]_0^1 + [x\ln x - x]_1^e$
	        $= ((1-1)^2 - (0-1)^2) + ((e\ln e - e) - (1\ln 1 - 1))$
	        $= (0 - 1) + ((e - e) - (0 - 1)) = -1 + 1 = 0$

[[换元法（解微分方程）]]

1. 第一步：利用解的结构求出$p(x)$     [[线性微分方程解的结构]]
	1.  **核心知识点**：对于一个线性非齐次微分方程$y' + p(x)y = q(x)$，如果$y_1$和$y_2$是它的任意两个解，那么它们的差$y_h = y_2 - y_1$一定是其对应的**齐次方程**$y' + p(x)y = 0$的一个解
	2. 因为$y_1$和$y_2$都是原方程的解，所以它们满足：[[线性方程的叠加原理]] 
	    $y_1' + p(x)y_1 = q(x)$  ---(1)
	    $y_2' + p(x)y_2 = q(x)$  ---(2)
		用(2)式减去(1)式，得到：
		$(y_2' - y_1') + p(x)(y_2 - y_1) = q(x) - q(x)$
		即：
		$(y_2 - y_1)' + p(x)(y_2 - y_1) = 0$
		这表明$y_h = y_2 - y_1$是齐次方程$y' + p(x)y = 0$的解
	3. 首先计算$y_2 - y_1$：
        $y_h = y_2 - y_1 = ((1+x^2)^2 + \sqrt{1+x^2}) - ((1+x^2)^2 - \sqrt{1+x^2}) = 2\sqrt{1+x^2}$
	4. 由于$y_h = 2\sqrt{1+x^2}$是齐次方程的解，那么它的任意常数倍也是解。为了计算方便，我们可以取$y_h^* = \frac{1}{2}y_h = \sqrt{1+x^2}$作为齐次方程的一个解
	5. 将$y_h^* = \sqrt{1+x^2}$代入齐次方程$y' + p(x)y = 0$。
        首先求$y_h^*$的导数：
        $(y_h^*)' = (\sqrt{1+x^2})' = \frac{1}{2\sqrt{1+x^2}} \cdot (1+x^2)' = \frac{2x}{2\sqrt{1+x^2}} = \frac{x}{\sqrt{1+x^2}}$
	6. 代入方程：
        $\frac{x}{\sqrt{1+x^2}} + p(x)(\sqrt{1+x^2}) = 0$
    7. 解出$p(x)$：
        $p(x)\sqrt{1+x^2} = -\frac{x}{\sqrt{1+x^2}}$
        $p(x) = -\frac{x}{(\sqrt{1+x^2})^2} = -\frac{x}{1+x^2}$
2. 第二步：利用任意一个特解求出$q(x)$ 
	1. 既然我们已经知道了$p(x)$，并且有特解$y_1$和$y_2$，我们可以任选一个（或者它们的线性组合，比如平均值）代入原方程$y' + p(x)y = q(x)$来求$q(x)$ 
	2. 简化计算，我们观察到$y_1$和$y_2$的平均值形式更简单 
		1. 令$y_p = \frac{y_1 + y_2}{2}$。
		    $y_p = \frac{((1+x^2)^2 - \sqrt{1+x^2}) + ((1+x^2)^2 + \sqrt{1+x^2})}{2} = \frac{2(1+x^2)^2}{2} = (1+x^2)^2$
	3. 由于$y_1$和$y_2$都是原方程的解，它们的平均值$y_p$也必然是原方程的解   
		1.  将$y_p = (1+x^2)^2$和$p(x) = -\frac{x}{1+x^2}$代入原方程$q(x) = y' + p(x)y$。
	    2.  首先求$y_p$的导数：
	        $y_p' = ((1+x^2)^2)' = 2(1+x^2) \cdot (1+x^2)' = 2(1+x^2) \cdot 2x = 4x(1+x^2)$
	    3.  代入方程：
	        $q(x) = y_p' + p(x)y_p = 4x(1+x^2) + (-\frac{x}{1+x^2})((1+x^2)^2)$
	    4.  化简：
	        $q(x) = 4x(1+x^2) - x(1+x^2)$
	        $q(x) = 3x(1+x^2)$
-  衍生 
	- [[一阶线性微分方程解的结构]] 
	-  本题中计算$(\sqrt{1+x^2})'$和$((1+x^2)^2)'$时都用到了 #链式法则
	- [[公式法（解一阶线性微分方程）]]  
		-  这是最基本的问题。给定$y' + p(x)y = q(x)$，要求你用**公式法**或**常数变易法**求出通解。
	    *   **公式法**：$y = e^{-\int p(x)dx} \left( \int q(x)e^{\int p(x)dx} dx + C \right)$。
	    *   **例题**：求解$y' - \frac{1}{x}y = x^2$的通解。
	* [[伯努利方程的形式与求解]]  
		*  **例题**：求解$y' + \frac{1}{x}y = xy^2$
	* 高阶线性微分方程解的结构 
		* 本题的原理可以完全推广到高阶。例如，对于二阶线性方程$y'' + p(x)y' + r(x)y = f(x)$，其通解结构也是$y = y_h + y_p$ 
		*  $y_h = c_1y_1 + c_2y_2$，其中$y_1, y_2$是对应齐次方程的两个线性无关的解。
	    *   如果$\phi_1, \phi_2$是非齐次方程的两个特解，则$\phi_1 - \phi_2$是对应齐次方程的解。
	    *   **例题**：已知$y_1 = e^x, y_2 = xe^x, y_3 = \sin x$是某二阶线性非齐次微分方程的三个解，求该方程。

[[间断点的类型和种类]] 
1. 第一步：求$f(0)$的值 #导数的定义   
	1. 当$x \le 0$时，$f(x)=x$。因此，$f(0)=0$ 
2. 第二步：求左导数$f'_{-}(0)$ 
	1. $f'_{-}(0) = \lim_{x \to 0^-} \frac{f(x) - f(0)}{x - 0}$
		当$x \to 0^-$时，$x$是从负数一侧趋近于0，所以$x<0$。根据函数定义，此时$f(x)=x$。
		代入公式：
		$f'_{-}(0) = \lim_{x \to 0^-} \frac{x - 0}{x} = \lim_{x \to 0^-} 1 = 1$
		所以，左导数存在且等于1。
3. 第三步：求右导数$f'_{+}(0)$
	$f'_{+}(0) = \lim_{x \to 0^+} \frac{f(x) - f(0)}{x - 0}$
	当$x \to 0^+$时，$x$是从正数一侧趋近于0，所以$x>0$。对于任何一个$x>0$，它必然会落入某个区间$(\frac{1}{n+1}, \frac{1}{n}]$中，此时$f(x) = \frac{1}{n}$。
	代入公式：
	$f'_{+}(0) = \lim_{x \to 0^+} \frac{f(x)}{x}$
	这里的难点在于$f(x)$的表达式依赖于$n$，而$n$又依赖于$x$
	[[夹逼准则（夹逼定理）]]
	1. 对于任意一个$x$满足$\frac{1}{n+1} < x \le \frac{1}{n}$，我们有$f(x)=\frac{1}{n}$ 
		1. 构造关于$\frac{f(x)}{x}$的不等式：
			1.  由$x \le \frac{1}{n}$，因为$x>0, n>0$，两边取倒数并乘以$f(x)=\frac{1}{n}$，得到 $\frac{f(x)}{x} = \frac{1/n}{x} \ge \frac{1/n}{1/n} = 1$。
			2.  由$\frac{1}{n+1} < x$，同理，得到 $\frac{f(x)}{x} = \frac{1/n}{x} < \frac{1/n}{1/(n+1)} = \frac{n+1}{n}$
		2. 得到了一个不等式：
			$1 \le \frac{f(x)}{x} < \frac{n+1}{n}$
		3. 考察极限过程。当$x \to 0^+$时，为了让$x$所在的区间$(\frac{1}{n+1}, \frac{1}{n}]$趋近于0，整数$n$必须趋向于无穷大，即$n \to \infty$
	2. 对不等式的两边取极限：
		左边：$\lim_{x \to 0^+} 1 = 1$
		右边：$\lim_{x \to 0^+} \frac{n+1}{n} = \lim_{n \to \infty} (1 + \frac{1}{n}) = 1$

#正态分布 
$X \sim N(\mu, \sigma^2)$。$\mu$ 是均值，$\sigma^2$ 是方差。
*   **概率密度函数 (PDF)**：$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$
*   **数字特征**：
    *   期望：$E(X) = \mu$
    *   方差：$D(X) = \sigma^2$
1. [[标准化正态分布]]
	1. $Z = \frac{X - \mu}{\sigma}$
2. 对题目中的概率表达式进行标准化
	1. 要求解的是 $p = P\{X \le \mu + \sigma^2\}$。为了使用标准正态分布来求解，我们需要对不等式 $X \le \mu + \sigma^2$ 的两边进行相同的变换，使其变成关于 $Z$ 的不等式 
		1. 第一步：不等式两边同时减去均值 $\mu$
			1. $X - \mu \le (\mu + \sigma^2) - \mu$ 
				1. $X - \mu \le \sigma^2$ 
		2. 第二步：不等式两边同时除以标准差 $\sigma$ 
			1. 题目给定 $\sigma > 0$，所以除以一个正数并不会改变不等号的方向。
		        $\frac{X - \mu}{\sigma} \le \frac{\sigma^2}{\sigma}$
		    2. 简化后得到：
		        $\frac{X - \mu}{\sigma} \le \sigma$
		3. 第三步：用 $Z$ 替换 
			1. 标准化公式，我们知道 $Z = \frac{X - \mu}{\sigma}$，所以原概率表达式可以写成：
		        $p = P\{Z \le \sigma\}$ 
3. 引入标准正态分布的累积分布函数 $\Phi(z)$  [[（累计）分布函数的定义]]  
	1. $\Phi(z) = P\{Z \le z\}$  
		1. 将我们的结果 $p = P\{Z \le \sigma\}$ 与 $\Phi(z)$ 的定义进行对比，可以发现，这里的 $z$ 就等于 $\sigma$。因此：
		    $p = \Phi(\sigma)$ 
	2. 分析 $p$ 与 $\sigma$ 的关系 
		1. 问题转化为了分析函数 $p = \Phi(\sigma)$ 的单调性 
			1. [[累积分布函数的性质]]  
				1. 它是一个**严格单调递增**的函数。这意味着，随着自变量 $z$ 的增大，函数值 $\Phi(z)$（也就是概率）也会随之增大  
	    同时，我们注意到，在整个推导过程中，参数 $\mu$ 被消掉了，最终结果 $p = \Phi(\sigma)$ 与 $\mu$ 无关。因此，$p$ 的值不随 $\mu$ 的变化而变化。
4.  **得出结论**
    *   $p$ 随着 $\sigma$ 的增加而增加。
    *   $p$ 不随 $\mu$ 的变化而变化。

		*   **期望**：$E(Y) = np = 2 \times \frac{1}{3} = \frac{2}{3}$。
		*   **方差**：$D(X) = np(1-p) = 2 \times \frac{1}{3} \times (1 - \frac{1}{3}) = 2 \times \frac{1}{3} \times \frac{2}{3} = \frac{4}{9}$。
		*   **方差**：$D(Y) = np(1-p) = 2 \times \frac{1}{3} \times \frac{2}{3} = \frac{4}{9}$
3. 第三步：计算协方差 Cov(X, Y) 
	1. [[皮尔逊相关系数]] 
		1. $\rho_{XY} = \frac{Cov(X,Y)}{\sqrt{DX}\sqrt{DY}}$ 
	2. 求出了分母中的 $DX$ 和 $DY$，现在需要求解分子，即协方差 $Cov(X,Y)$。
		协方差的计算公式为：
		$Cov(X,Y) = E(XY) - E(X)E(Y)$
	3. 已经有 $E(X)$ 和 $E(Y)$，所以关键是求出 $E(XY)$，即乘积 $XY$ 的期望。为此，我们需要知道 $XY$ 的概率分布 
4. 第四步：求解 E(XY) 
	1. 分析 $X$ 和 $Y$ 的取值 
		1. $X$ 和 $Y$ 是2次试验中事件发生的次数，所以它们的可能取值都是 $0, 1, 2$
		2. 但是，$X$ 和 $Y$ 不是独立的，因为它们是对同两次试验的计数
		3. 例如，如果第一次试验结果是 $A_1$，那么它就不可能是 $A_2$。它们之间存在一个重要的约束关系：$X+Y \le 2$ 
			1. 因为总共只有2次试验， $A_1$ 发生的次数加上 $A_2$ 发生的次数，最多等于2（当两次试验结果分别是 $A_1$ 和 $A_2$ 时，或两次都是 $A_1$ 或两次都是 $A_2$ 时）
	2. 分析乘积 $XY$ 的取值  
		1. 基于 $X, Y \in \{0, 1, 2\}$ 和 $X+Y \le 2$，我们来看 $XY$ 的可能值：
		    *   如果 $X=0$ 或 $Y=0$，则 $XY=0$。
		    *   如果 $X=1, Y=1$，则 $XY=1$。
		    *   $X=2$ 时，必然 $Y=0$，所以 $XY=0$。
		    *   $Y=2$ 时，必然 $X=0$，所以 $XY=0$。
		    *   $X=1, Y=2$ 或 $X=2, Y=1$ 等情况是不可能的，因为 $X+Y > 2$。
		    所以，$XY$ 的可能取值只有 $0$ 和 $1$。
	3. 计算 $XY$ 的概率分布 
		1. 只需要计算 $P(XY=1)$ 的概率，那么 $P(XY=0)$ 自然就是 $1 - P(XY=1)$。
		    事件 $\{XY=1\}$ 发生的唯一情况是 $\{X=1, Y=1\}$
		2. $\{X=1, Y=1\}$ 表示在2次试验中，结果 $A_1$ 发生1次，结果 $A_2$ 也发生1次。这有两种可能的结果序列：
		    *   第一次是 $A_1$，第二次是 $A_2$。概率为 $P(A_1) \times P(A_2) = \frac{1}{3} \times \frac{1}{3} = \frac{1}{9}$。
		    *   第一次是 $A_2$，第二次是 $A_1$。概率为 $P(A_2) \times P(A_1) = \frac{1}{3} \times \frac{1}{3} = \frac{1}{9}$。	
		3. 因此，$P(XY=1) = P(X=1, Y=1) = \frac{1}{9} + \frac{1}{9} = \frac{2}{9}$。
		    （这里也可以用多项分布的概率公式计算：$P(X=1, Y=1) = \frac{2!}{1!1!0!} (\frac{1}{3})^1 (\frac{1}{3})^1 (\frac{1}{3})^0 = 2 \times \frac{1}{9} = \frac{2}{9}$）
		4. 于是，$P(XY=0) = 1 - P(XY=1) = 1 - \frac{2}{9} = \frac{7}{9}$ 
	4. 计算 $E(XY)$ 
		1. 根据期望的定义：
		    $E(XY) = \sum_{k} k \cdot P(XY=k) = 0 \cdot P(XY=0) + 1 \cdot P(XY=1) = 0 \times \frac{7}{9} + 1 \times \frac{2}{9} = \frac{2}{9}$ 
	5. 第五步：完成最终计算
		1. **计算协方差**：
	    $Cov(X,Y) = E(XY) - E(X)E(Y) = \frac{2}{9} - (\frac{2}{3}) \times (\frac{2}{3}) = \frac{2}{9} - \frac{4}{9} = -\frac{2}{9}$。
		2. **计算相关系数**：
	    $\rho_{XY} = \frac{Cov(X,Y)}{\sqrt{DX}\sqrt{DY}} = \frac{-2/9}{\sqrt{4/9}\sqrt{4/9}} = \frac{-2/9}{ (2/3) \times (2/3) } = \frac{-2/9}{4/9} = -\frac{1}{2}$。

	-  **改变 $k$ 的取值范围**

	    *   **若 $k>1$**：此时特征方程的判别式 $\Delta = 4-4k < 0$，得到一对共轭复根 $r = -1 \pm i\sqrt{k-1}$。通解为 $y(x) = e^{-x}(C_1\cos(\sqrt{k-1}x) + C_2\sin(\sqrt{k-1}x))$。由于 $e^{-x}$ 这一项的存在，当 $x \to +\infty$ 时，$y(x)$ 和 $y'(x)$ 仍然趋于0，反常积分依然收敛，最终结果也仍然是 $\frac{3}{k}$。这说明这个巧妙的解法普适性很强
	*  **非齐次方程**：将原方程改为 $y''+2y'+ky=f(x)$，例如 $f(x)=e^{-x}$ 或 $f(x)=\sin(x)$。这就需要先求特解，再求通解

- 这道题目要求我们计算一个与路径终点相关的曲线积分$I(t)$，并求其最小值。 
#分段光滑曲线的积分 [[格林公式]]  
[[第二类曲线积分与路径无关的条件]]
1. 第一步：求原函数 $f(x,y)$  
	1. 对偏导数进行积分  
		1. $f(x,y) = \int \frac{\partial f(x,y)}{\partial x} dx = \int (2x+1)e^{2x-y} dx$ 
		2. 为了计算这个积分，我们可以把$e^{-y}$看作常数提出来：
		    $f(x,y) = e^{-y} \int (2x+1)e^{2x} dx$
		    这个积分可以使用 #分部积分法 $\int u dv = uv - \int v du$ 来求解。
		    令 $u = 2x+1$，$dv = e^{2x}dx$。则 $du = 2dx$，$v = \frac{1}{2}e^{2x}$
		$\int (2x+1)e^{2x} dx = (2x+1)(\frac{1}{2}e^{2x}) - \int (\frac{1}{2}e^{2x})(2dx) =$
		$(x+\frac{1}{2})e^{2x} - \int e^{2x}dx = (x+\frac{1}{2})e^{2x} - \frac{1}{2}e^{2x} = xe^{2x}$ 
		将结果代回，得到：
	    $f(x,y) = e^{-y}(xe^{2x}) + C(y) = xe^{2x-y} + C(y)$
	2. 利用边界条件确定$C(y)$ 
		1. 题目给出了条件 $f(0,y) = y+1$。我们将$x=0$代入上面求得的$f(x,y)$表达式中：
		    $f(0,y) = 0 \cdot e^{2(0)-y} + C(y) = C(y)$
		    因此，我们得到 $C(y) = y+1$
	3. **写出$f(x,y)$的完整表达式**：
	    将$C(y)$代回，我们得到原函数：
	    $f(x,y) = xe^{2x-y} + y + 1$
2. 第二步：计算曲线积分 $I(t)$
	1. 题目要求计算的曲线积分是 $I(t) = \int_{L_t} \frac{\partial f(x,y)}{\partial x} dx + \frac{\partial f(x,y)}{\partial y} dy$
	2. 识别积分形式 
		1. 这个积分的形式是 $\int_L Pdx + Qdy$，其中 $P = \frac{\partial f}{\partial x}$ 且 $Q = \frac{\partial f}{\partial y}$。这正是函数$f(x,y)$的全微分$df$的线积分 
	3. [[曲线积分基本定理]] 在本题中，路径$L_t$的起点是$A=(0,0)$，终点是$B=(1,t)$。因此：
		    $I(t) = f(1,t) - f(0,0)$
	4. 代入计算
		1. 使用我们第一步求出的$f(x,y) = xe^{2x-y} + y + 1$：
		    $f(1,t) = 1 \cdot e^{2(1)-t} + t + 1 = e^{2-t} + t + 1$
		    $f(0,0) = 0 \cdot e^{2(0)-0} + 0 + 1 = 1$
		    所以，
		    $I(t) = (e^{2-t} + t + 1) - 1 = e^{2-t} + t$
3. 第三步：求 $I(t)$ 的最小值
	1. 问题转化为了求一元函数 $I(t) = e^{2-t} + t$ 的最小值
		1. **求导数**：
		    对$I(t)$求关于$t$的导数：
		    $I'(t) = \frac{d}{dt}(e^{2-t} + t) = e^{2-t} \cdot (-1) + 1 = -e^{2-t} + 1$
		2. **求驻点**：
		    令导数等于零，寻找可能的极值点：
		    $I'(t) = 0 \implies -e^{2-t} + 1 = 0 \implies e^{2-t} = 1$
		    两边取自然对数：
		    $\ln(e^{2-t}) = \ln(1) \implies 2-t = 0 \implies t=2$
	2. 判断极值类型
		1. 我们可以使用一阶导数法来判断单调性：
		    *   当 $t<2$ 时，$2-t>0$，所以 $e^{2-t} > 1$。因此 $I'(t) = 1 - e^{2-t} < 0$，$I(t)$在此区间单调递减。
		    *   当 $t>2$ 时，$2-t<0$，所以 $0 < e^{2-t} < 1$。因此 $I'(t) = 1 - e^{2-t} > 0$，$I(t)$在此区间单调递增
		2. 函数在$t=2$处由减变增，因此在$t=2$处取得极小值，同时也是全局最小值  
	3. **计算最小值**：
	    将$t=2$代入$I(t)$的表达式：
	    $I_{min} = I(2) = e^{2-2} + 2 = e^0 + 2 = 1 + 2 = 3$

	- #保守场的判定  [[第二类曲线积分与路径无关的条件]]
		- 一个常见的考点是判断给定的积分$\int Pdx+Qdy$是否与路径无关。在单连通区域内，其充要条件是$\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$。题目可能会直接让你判断，或者让你计算一个沿复杂路径（如折线、圆弧）的积分，此时先判断其是否与路径无关，如果是，就可以选择一条更简单的路径（如直线段）来计算。  
	- [[格林公式]]  
		-     格林公式建立了平面区域上的二重积分与沿该区域边界的曲线积分之间的关系：$\oint_L Pdx+Qdy = \iint_D (\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y})dA$。
		    *   如果题目中的路径$L$是一条**封闭曲线**，通常会考虑使用格林公式。
		    *   特别地，如果$\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$，那么沿任何封闭曲线的积分都为0，这也再次验证了保守场的性质。
	- #非保守场的曲线积分计算 
		- 如果积分与路径有关（即$\frac{\partial P}{\partial y} \neq \frac{\partial Q}{\partial x}$），题目会明确给出积分路径的方程（如$y=x^2$）。此时的解法是：

		    *   **代入化简**：将$x, y, dx, dy$全部用参数$t$和$dt$表示，将曲线积分转化为对参数$t$的定积分。
		    *   **计算定积分**。
	- #利用曲线积分求面积 
		- 格林公式的一个巧妙应用是计算平面区域$D$的面积$A = \iint_D dA$。可以选择合适的$P, Q$使得$\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = 1$，然后通过计算边界曲线积分来求面积。常用的公式有：
