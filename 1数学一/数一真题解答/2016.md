#积分收敛 
#含有多个瑕点的反常积分收敛的充要条件 
1.   第一部分：分析 $\int_{0}^{1} \frac{1}{x^a(1+x)^b} dx$ 的收敛性
	1. 积分的瑕点（可能导致积分发散的点）是$x=0$ 
		1. #极限比较判别法 来判断其收敛性
			1. 确定被积函数和比较函数
				1. 被积函数是 $f(x) = \frac{1}{x^a(1+x)^b}$。
				    *   当$x \to 0^+$时，$(1+x)$项趋近于$1$，所以$(1+x)^b$趋近于$1^b=1$。此时，$f(x)$的行为主要由$x^a$决定。因此，我们选择一个更简单的函数$g(x) = \frac{1}{x^a}$作为比较函数
				2.  **计算极限**：
				    $\lim_{x \to 0^+} \frac{f(x)}{g(x)} = \lim_{x \to 0^+} \frac{\frac{1}{x^a(1+x)^b}}{\frac{1}{x^a}} = \lim_{x \to 0^+} \frac{1}{(1+x)^b} = \frac{1}{(1+0)^b} = 1$
				    因为极限值$1$是一个不为0的常数，所以$\int_{0}^{1} \frac{1}{x^a(1+x)^b} dx$与$\int_{0}^{1} \frac{1}{x^a} dx$的敛散性相同。
			2.  **判断比较函数的收敛性**：
			    $\int_{0}^{1} \frac{1}{x^a} dx$是p-积分的一种形式。
			    **核心知识点：p-积分在0点的敛散性**
			    反常积分$\int_{0}^{c} \frac{1}{x^p} dx$ (其中$c>0$)，当$p<1$时收敛，当$p \ge 1$时发散。
			    因此，为了使$\int_{0}^{1} \frac{1}{x^a} dx$收敛，必须满足$a<1$。
2. 第二部分：分析 $\int_{1}^{+\infty} \frac{1}{x^a(1+x)^b} dx$ 的收敛性
	1. 这部分积分是无穷区间上的反常积分。我们同样使用 #极限比较判别法 
		1.  **确定被积函数和比较函数**：
	    *   被积函数是 $f(x) = \frac{1}{x^a(1+x)^b}$。
	    *   当$x \to +\infty$时，$(1+x)$中的常数$1$可以忽略不计，所以$(1+x) \sim x$。因此，分母$x^a(1+x)^b \sim x^a \cdot x^b = x^{a+b}$。我们选择比较函数$g(x) = \frac{1}{x^{a+b}}$。
	2. 计算极限
		1. $\lim_{x \to +\infty} \frac{f(x)}{g(x)} = \lim_{x \to +\infty} \frac{\frac{1}{x^a(1+x)^b}}{\frac{1}{x^{a+b}}} =$
			$\lim_{x \to +\infty} \frac{x^{a+b}}{x^a(1+x)^b} = \lim_{x \to +\infty} (\frac{x}{1+x})^b = (\lim_{x \to +\infty} \frac{1}{\frac{1}{x}+1})^b = 1^b = 1$
		2. 因为极限值$1$是一个不为0的常数，所以$\int_{1}^{+\infty} \frac{1}{x^a(1+x)^b} dx$与$\int_{1}^{+\infty} \frac{1}{x^{a+b}} dx$的敛散性相同
	3.  **判断比较函数的收敛性**：
	    $\int_{1}^{+\infty} \frac{1}{x^{a+b}} dx$是p-积分的另一种形式。
	    **核心知识点：p-积分在无穷远点的敛散性**
	    反常积分$\int_{c}^{+\infty} \frac{1}{x^p} dx$ (其中$c>0$)，当$p>1$时收敛，当$p \le 1$时发散。
	    因此，为了使$\int_{1}^{+\infty} \frac{1}{x^{a+b}} dx$收敛，必须满足指数$a+b>1$。
	
	**结论2**：第二部分积分收敛的条件是$a+b>1$。
为了使原积分$I = \int_{0}^{+\infty} \frac{1}{x^a(1+x)^b} dx$收敛，必须同时满足第一部分和第二部分的收敛条件。
即：$a<1$ 且 $a+b>1$
	
- 衍生  
	- [[p-积分]]  不同形式的 #p-积分判别
	    *   $\int_{c}^{+\infty} \frac{1}{x^p} dx$：要求$p>1$收敛。
	    *   $\int_{0}^{c} \frac{1}{x^p} dx$：要求$p<1$收敛。
	    *   $\int_{a}^{b} \frac{1}{(x-a)^p} dx$：瑕点在$a$，要求$p<1$收敛。
	    *   $\int_{a}^{b} \frac{1}{(b-x)^p} dx$：瑕点在$b$，要求$p<1$收敛。
	-   更复杂的被积函数 
		- 例如，判断$\int_{0}^{+\infty} \frac{\ln(1+x)}{x^p} dx$的敛散性。
	        *   在$x \to 0^+$时，$\ln(1+x) \sim x$，所以被积函数 $\sim \frac{x}{x^p} = \frac{1}{x^{p-1}}$。要求$p-1<1$，即$p<2$。
	        *   在$x \to +\infty$时，被积函数与$\frac{\ln x}{x^p}$同敛散。根据p-积分的对数形式，要求$p>1$。
	        *   综合起来，收敛条件是$1<p<2$。
	-  与 #Gamma函数 和 #Beta函数 的关系  [[特殊函数(伽马Gamma函数)]]
		- 本题的积分实际上是**Beta函数**的一种形式。Beta函数的定义之一是 $B(m, n) = \int_{0}^{+\infty} \frac{t^{m-1}}{(1+t)^{m+n}} dt$ 
		-  我们对比题中积分 $\int_{0}^{+\infty} \frac{1}{x^a(1+x)^b} dx$ 和Beta函数的标准形式 $B(p, q) = \int_0^1 t^{p-1}(1-t)^{q-1} dt$。通过变量代换 $x = \frac{t}{1-t}$，可以证明原积分等于 $B(1-a, a+b-1)$
		- #Beta函数收敛域 
			- Beta函数$B(p, q)$收敛的条件是$p>0$且$q>0$ 
			- 因此，我们得到$1-a>0$且$a+b-1>0$。
		    *   这直接给出了$a<1$和$a+b>1$的结论。这是一个更高等但更快捷的方法
	- #绝对收敛和条件收敛 
		对于非正项函数的反常积分，如$\int_{1}^{+\infty} \frac{\sin x}{x^p} dx$，需要讨论绝对收敛 ($\int |\frac{\sin x}{x^p}| dx$收敛)和条件收敛($\int \frac{\sin x}{x^p} dx$收敛但$\int |\frac{\sin x}{x^p}| dx$发散)。
	    *   $\int_{1}^{+\infty} |\frac{\sin x}{x^p}| dx \le \int_{1}^{+\infty} \frac{1}{x^p} dx$。当$p>1$时，$\int_{1}^{+\infty} \frac{1}{x^p} dx$收敛，所以原积分绝对收敛。
	    *   当$0<p \le 1$时，可以证明积分条件收敛（通常使用狄利克雷判别法）。

 #求原函数   [[原函数]] 
#乘法法则   #原函数必须连续
- 解法1：不定积分法 + 保证连续性 
1. 第一步：对每一段分别求不定积分 
	1. $F(x) = \int \ln x dx = x\ln x - \int x \cdot \frac{1}{x}dx = x\ln x - x + C_2$
		1. $F(x) = \int 2(x-1)dx = (x-1)^2 + C_1$ 
		2. 得到了原函数$F(x)$的通用表达式：
		$F(x)=\begin{cases} (x-1)^2 + C_1, & x<1 \\ x\ln x - x + C_2, & x\geq 1 \end{cases}$
2. 第二步：利用分段点的连续性确定常数关系
	1. $F(x)$必须在分段点$x=1$处连续。这意味着$F(x)$在$x=1$处的左极限必须等于右极限（也等于函数值）。
		*   计算左极限：
		    $\lim_{x\to 1^-} F(x) = \lim_{x\to 1^-} [(x-1)^2 + C_1] = (1-1)^2 + C_1 = C_1$
		*   计算右极限（即函数值）：
		    $\lim_{x\to 1^+} F(x) = F(1) = 1\cdot\ln 1 - 1 + C_2 = 0 - 1 + C_2 = -1 + C_2$
		令左极限等于右极限：
		$C_1 = -1 + C_2$
	2. 令$C_1=0$。
		根据$C_1 = -1 + C_2$，可得$C_2 = C_1 + 1 = 0 + 1 = 1$
	3. 将$C_1=0, C_2=1$代入$F(x)$的表达式中，得到一个特解：
		$F(x)=\begin{cases} (x-1)^2, & x<1 \\ x\ln x - x + 1, & x\geq 1 \end{cases}$
	4. 将$x\geq 1$的部分进行因式分解，可以写成$x(\ln x - 1) + 1$。
		所以，$F(x)=\begin{cases} (x-1)^2, & x<1 \\ x(\ln x - 1) + 1, & x\geq 1 \end{cases}$
- 解法2： #变上限积分法 
	- 第一步：构造变上限积分函数  
		- 根据牛顿-莱布尼茨公式，函数$F(x) = \int_a^x f(t)dt$是$f(x)$的一个原函数，并且满足$F(a)=0$。这个函数是天然连续的 
		- 计算方便，我们通常选取分段点作为积分下限$a$。这里，我们取$a=1$。
			令$F(x) = \int_1^x f(t)dt$ 
	2. 第二步：分情况计算定积分 
		1. 当$x<1$时，积分上限在积分下限的左边：
	    $F(x) = \int_1^x f(t)dt = \int_1^x 2(t-1)dt = [(t-1)^2]_1^x = (x-1)^2 - (1-1)^2 = (x-1)^2$
		*   当$x\geq 1$时，积分路径上的$t$始终大于等于1：
	    $F(x) = \int_1^x f(t)dt = \int_1^x \ln t dt = [t\ln t - t]_1^x$
	    $= (x\ln x - x) - (1\ln 1 - 1) = x\ln x - x - (0-1) = x\ln x - x + 1$
	1. 第三步：组合成最终结果
	将两段的结果合并，得到：
	$F(x)=\begin{cases} (x-1)^2, & x<1 \\ x\ln x - x + 1, & x\geq 1 \end{cases}$
-  衍生 
	- 原函数的 #可导性 问题
		-  **结论**：如果$f(x)$在某点$x_0$处连续，那么它的原函数$F(x)$在该点一定可导，且$F'(x_0) = f(x_0)$ 
		-   **在本题中**：我们需要判断$f(x)$在$x=1$处是否连续。
	        $\lim_{x\to 1^-} f(x) = \lim_{x\to 1^-} 2(x-1) = 0$
	        $\lim_{x\to 1^+} f(x) = \lim_{x\to 1^+} \ln x = \ln 1 = 0$
	        因为左右极限相等且等于$f(1)=0$，所以$f(x)$在$x=1$处连续。因此，它的原函数$F(x)$在$x=1$处是**可导的**，且$F'(1)=f(1)=0$
	- 原函数的 #二阶可导性 问题
		-  **结论**：原函数$F(x)$的二阶可导性取决于$f(x)$的一阶可导性，即$F''(x) = f'(x)$。
	    *   **在本题中**：我们需要判断$f(x)$在$x=1$处是否可导。
	        *   当$x<1$时，$f'(x) = 2$。所以左导数$f'_-(1)=2$。
	        *   当$x>1$时，$f'(x) = \frac{1}{x}$。所以右导数$f'_+(1)=1$。
	        因为左右导数不相等，所以$f(x)$在$x=1$处**不可导**。
	        *   **衍生结论**：这意味着原函数$F(x)$在$x=1$处是**二阶不可导**的。
	-  **求包含分段点的定积分**
	    *   例如，计算$\int_0^e f(x)dx$。
	    *   需要将积分区间根据分段点拆开：
	        $\int_0^e f(x)dx = \int_0^1 f(x)dx + \int_1^e f(x)dx$
	        $= \int_0^1 2(x-1)dx + \int_1^e \ln x dx$
	        $= [(x-1)^2]_0^1 + [x\ln x - x]_1^e$
	        $= ((1-1)^2 - (0-1)^2) + ((e\ln e - e) - (1\ln 1 - 1))$
	        $= (0 - 1) + ((e - e) - (0 - 1)) = -1 + 1 = 0$

[[换元法（解微分方程）]]

1. 第一步：利用解的结构求出$p(x)$     [[线性微分方程解的结构]]
	1.  **核心知识点**：对于一个线性非齐次微分方程$y' + p(x)y = q(x)$，如果$y_1$和$y_2$是它的任意两个解，那么它们的差$y_h = y_2 - y_1$一定是其对应的**齐次方程**$y' + p(x)y = 0$的一个解
	2. 因为$y_1$和$y_2$都是原方程的解，所以它们满足：[[线性方程的叠加原理]] 
	    $y_1' + p(x)y_1 = q(x)$  ---(1)
	    $y_2' + p(x)y_2 = q(x)$  ---(2)
		用(2)式减去(1)式，得到：
		$(y_2' - y_1') + p(x)(y_2 - y_1) = q(x) - q(x)$
		即：
		$(y_2 - y_1)' + p(x)(y_2 - y_1) = 0$
		这表明$y_h = y_2 - y_1$是齐次方程$y' + p(x)y = 0$的解
	3. 首先计算$y_2 - y_1$：
        $y_h = y_2 - y_1 = ((1+x^2)^2 + \sqrt{1+x^2}) - ((1+x^2)^2 - \sqrt{1+x^2}) = 2\sqrt{1+x^2}$
	4. 由于$y_h = 2\sqrt{1+x^2}$是齐次方程的解，那么它的任意常数倍也是解。为了计算方便，我们可以取$y_h^* = \frac{1}{2}y_h = \sqrt{1+x^2}$作为齐次方程的一个解
	5. 将$y_h^* = \sqrt{1+x^2}$代入齐次方程$y' + p(x)y = 0$。
        首先求$y_h^*$的导数：
        $(y_h^*)' = (\sqrt{1+x^2})' = \frac{1}{2\sqrt{1+x^2}} \cdot (1+x^2)' = \frac{2x}{2\sqrt{1+x^2}} = \frac{x}{\sqrt{1+x^2}}$
	6. 代入方程：
        $\frac{x}{\sqrt{1+x^2}} + p(x)(\sqrt{1+x^2}) = 0$
    7. 解出$p(x)$：
        $p(x)\sqrt{1+x^2} = -\frac{x}{\sqrt{1+x^2}}$
        $p(x) = -\frac{x}{(\sqrt{1+x^2})^2} = -\frac{x}{1+x^2}$
2. 第二步：利用任意一个特解求出$q(x)$ 
	1. 既然我们已经知道了$p(x)$，并且有特解$y_1$和$y_2$，我们可以任选一个（或者它们的线性组合，比如平均值）代入原方程$y' + p(x)y = q(x)$来求$q(x)$ 
	2. 简化计算，我们观察到$y_1$和$y_2$的平均值形式更简单 
		1. 令$y_p = \frac{y_1 + y_2}{2}$。
		    $y_p = \frac{((1+x^2)^2 - \sqrt{1+x^2}) + ((1+x^2)^2 + \sqrt{1+x^2})}{2} = \frac{2(1+x^2)^2}{2} = (1+x^2)^2$
	3. 由于$y_1$和$y_2$都是原方程的解，它们的平均值$y_p$也必然是原方程的解

*   **概率密度函数 (PDF)**：$f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}$
*   **数字特征**：
    *   期望：$E(X) = \mu$
    *   方差：$D(X) = \sigma^2$
