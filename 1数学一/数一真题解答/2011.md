1. #拐点   [[重数与导数之间的关系]]  [[拐点（微积分pan bie
	如果函数 $f(x)$ 可以表示为 $f(x)=(x-a)^n g(x)$，其中 $n$ 是一个正整数，且 $g(a) \neq 0$ (这意味着 $x=a$ 是 $f(x)$ 的一个 **$n$ 重根**)，那么：
	$f(x)$ 在 $x=a$ 处的前 $n-1$ 阶导数均为零，而第 $n$ 阶导数不为零。
	即：
	$f(a) = f'(a) = f''(a) = \dots = f^{(n-1)}(a) = 0$
	$f^{(n)}(a) \neq 0$
2.  对于根 $x=1$ 
    *   它是 $1$ 重根 ($n=1$)。
    *   根据上述理论，$f(1)=0$，但 $f'(1) \neq 0$。
    *   因为 $f''(1)$ 不一定为零，所以 $(1,0)$ 一般不是拐点。
3. 对于根 $x=2$ 
    *   它是 $2$ 重根 ($n=2$)。
    *   根据理论，$f(2)=0, f'(2)=0$，但 $f''(2) \neq 0$。
    *   因为 $f''(2) \neq 0$，不满足拐点的必要条件，所以 $(2,0)$ 不是拐点。实际上，这是一个**局部极值点**。
4. 对于根 $x=3$
    *   它是 $3$ 重根 ($n=3$)。
    *   根据理论，$f(3)=0, f'(3)=0, f''(3)=0$，但 $f'''(3) \neq 0$。
    *   这里满足了拐点的必要条件 $f''(3)=0$。现在我们需要判断充分条件，即 $f''(x)$ 在 $x=3$ 两侧的符号是否改变。
    *   我们可以利用泰勒展开的思想来理解。在 $x=3$ 附近，$f''(x)$ 的行为主要由它在 $x=3$ 处的第一个非零导数项决定。
        $f''(x) \approx \frac{f'''(3)}{1!}(x-3)$
    *   因为 $f'''(3) \neq 0$，当 $x$ 从小于3变到大于3时，$(x-3)$ 的符号从负变到正，因此 $f''(x)$ 的符号也随之改变。
    *   所以，$(3,0)$ 是一个拐点。
5. 对于根 $x=4$
    *   它是 $4$ 重根 ($n=4$)。
    *   根据理论，$f(4)=0, f'(4)=0, f''(4)=0, f'''(4)=0$，但 $f^{(4)}(4) \neq 0$。
    *   这里也满足了拐点的必要条件 $f''(4)=0$。我们来判断充分条件。
    *   在 $x=4$ 附近，$f''(x)$ 的行为主要由它在 $x=4$ 处的第一个非零导数项决定。
        $f''(x) \approx \frac{f^{(4)}(4)}{2!}(x-4)^2$
    *   因为 $f^{(4)}(4) \neq 0$ 且 $(x-4)^2 \ge 0$，所以 $f''(x)$ 在 $x=4$ 的两侧符号相同（都为正或都为负），没有发生改变。
    *   所以，$(4,0)$ 不是拐点。（它也是一个局部极值点）。
结论 
	只有点 $(3,0)$ 满足拐点的充要条件。因此，正确答案是 **(C)**

1. #定积分的保序性 [[定积分的性质]]
	1. 如果在一个区间 $[a, b]$ 上，有两个函数 $f(x)$ 和 $g(x)$ 满足 $f(x) \le g(x)$，那么它们的定积分也满足同样的大小关系：
	    $\int_{a}^{b} f(x) \, dx \le \int_{a}^{b} g(x) \, dx$
	    更进一步，如果在区间 $(a, b)$ 内恒有 $f(x) < g(x)$（允许在端点处相等），那么积分结果也是严格小于的：
	    $\int_{a}^{b} f(x) \, dx < \int_{a}^{b} g(x) \, dx$
2. 确定积分区间并比较被积函数的参数
	分区间是 $x \in [0, \frac{\pi}{4}]$。我们主要关注开区间 $(0, \frac{\pi}{4})$ 内的情况。
    在这个区间内，我们来比较 $\sin x$, $\cos x$ 和 $\cot x$ 的大小。
    *   **比较 $\sin x$ 和 $\cos x$**：在 #单位圆 上，当角度 $x$ 从 $0$ 增加到 $\frac{\pi}{4}$ 时，终边始终在直线 $y=x$ 的下方。这意味着 $y$ 坐标（$\sin x$）小于 $x$ 坐标（$\cos x$）。因此，对于任意 $x \in (0, \frac{\pi}{4})$，我们有 $\sin x < \cos x$。
    *   **比较 $\cos x$ 和 $\cot x$**：我们知道 $\cot x = \frac{\cos x}{\sin x}$。因为在 $(0, \frac{\pi}{4})$ 区间内, $0 < \sin x < \frac{\sqrt{2}}{2} < 1$。用一个正数 $\cos x$ 除以一个小于1的正数 $\sin x$，得到的结果必然大于 $\cos x$。所以，$\cot x > \cos x$。
	将以上两点结合，我们得到在区间 $x \in (0, \frac{\pi}{4})$ 内，三个函数参数的大小关系为：
    $\sin x < \cos x < \cot x$

3. 第三步：代入求值
	1. 将点$(x, y) = (0, 2)$代入第二步得到的表达式中。

- 考察了求解空间第二类曲线积分的三种核心方法。下面我们逐一详细解析题目中给出的三种解法 [[曲线积分，曲面积分类型]] 
解法 1: 降维为平面 #第二类曲线积分 (利用 #格林公式) [[格林公式]]
-  利用曲线所在的曲面方程，将三维的曲线积分降维成二维的平面曲线积分，然后使用格林公式求解 
1. 降维 
	1. $L$ 上的点同时满足 $x^2+y^2=1$ 和 $z=x+y$。我们可以将 $z$ 和 $dz$ 用 $x, y$ 和它们的微分来表示
	2. $z=x+y$，可得 $dz = dx+dy$ 
		1. $I = \oint_L xzdx + xdy + \frac{y^2}{2}dz$ 
		2. $= \oint_L x(x+y)dx + xdy + \frac{y^2}{2}(dx+dy)$
2. 整理为平面曲线积分形式
	1. 将上式中的 $dx$ 和 $dy$ 项合并，得到一个在 $xOy$ 平面上的投影曲线 $l: x^2+y^2=1$ 上的 #第二类曲线积分 
	2. $I = \oint_l (x(x+y) + \frac{y^2}{2})dx + (x + \frac{y^2}{2})dy$
	3.  $= \oint_l (x^2+xy+\frac{y^2}{2})dx + (x+\frac{y^2}{2})dy$
	4. #积分路径 $l$ 是 $xOy$ 平面上的 #单位圆 $x^2+y^2=1$。题目描述的定向“从z轴正向往z轴负向看去为逆时针方向”，正好对应了 $xOy$ 平面上单位圆的逆时针正方向
3. 应用 #格林公式 
	1. $\oint_l Pdx + Qdy = \iint_D (\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y})dxdy$ 
		1. 其中 $D$ 是由 $l$ 围成的区域，即单位圆盘 $x^2+y^2 \le 1$
		2.  令 $P(x,y) = x^2+xy+\frac{y^2}{2}$ 和 $Q(x,y) = x+\frac{y^2}{2}$
			计算偏导数：
		    $\frac{\partial Q}{\partial x} = \frac{\partial}{\partial x}(x+\frac{y^2}{2}) = 1$
		    $\frac{\partial P}{\partial y} = \frac{\partial}{\partial y}(x^2+xy+\frac{y^2}{2}) = x+y$
		    所以，$\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = 1 - (x+y) = 1-x-y$。
4. 计算二重积分
	1. $I = \iint_D (1-x-y)dxdy$ 
	由于积分区域 $D$ 是一个单位圆盘，使用极坐标计算更方便。令 $x=r\cos\theta, y=r\sin\theta$，则 $dxdy=rdrd\theta$。积分区域变为 $0 \le r \le 1, 0 \le \theta \le 2\pi$。
	    $I = \int_0^{2\pi} \int_0^1 (1-r\cos\theta-r\sin\theta)rdrd\theta$
	    $= \int_0^{2\pi} \int_0^1 (r-r^2\cos\theta-r^2\sin\theta)drd\theta$
	    $= \int_0^{2\pi} [\frac{r^2}{2} - \frac{r^3}{3}\cos\theta - \frac{r^3}{3}\sin\theta]_0^1 d\theta$
	    $= \int_0^{2\pi} (\frac{1}{2} - \frac{1}{3}\cos\theta - \frac{1}{3}\sin\theta) d\theta$
	    $= [\frac{1}{2}\theta - \frac{1}{3}\sin\theta + \frac{1}{3}\cos\theta]_0^{2\pi}$
    $= (\frac{1}{2}(2\pi) - 0 + \frac{1}{3}) - (0 - 0 + \frac{1}{3}) = \pi$
- 解法 2: #参数方程 法 
1. 参数化曲线 L 
	1. 曲线 $L$ 在 $xOy$ 平面上的投影是单位圆 $x^2+y^2=1$。根据其逆时针方向，可以参数化为：
	    $x = \cos t$
	    $y = \sin t$
	2.  $t$ 从 $0$ 到 $2\pi$ 
		1.  $x, y$ 代入平面方程 $z=x+y$，得到 $z$ 的参数方程：
	    $z = \cos t + \sin t$
	3.  $L$ 的参数方程为 $\vec{r}(t) = (\cos t, \sin t, \cos t + \sin t)$，$t \in [0, 2\pi]$ 
2. 计算微分 
    $dx = d(\cos t) = -\sin t dt$
    $dy = d(\sin t) = \cos t dt$
    $dz = d(\cos t + \sin t) = (-\sin t + \cos t) dt$ 
 3. 代入积分并计算
	 1. $I = \int_0^{2\pi} [(\cos t)(\cos t + \sin t)(-\sin t) + (\cos t)(\cos t) + \frac{\sin^2 t}{2}(-\sin t + \cos t)] dt$ 
	 2.     $(-\cos^2 t \sin t - \cos t \sin^2 t) + \cos^2 t + (-\frac{1}{2}\sin^3 t + \frac{1}{2}\sin^2 t \cos t)$
	    $= \cos^2 t - \cos^2 t \sin t - \frac{1}{2}\cos t \sin^2 t - \frac{1}{2}\sin^3 t$
	3. 因为其他项在一个周期 $[0, 2\pi]$ 上的积分都为零
		*   $\int_0^{2\pi} (-\cos^2 t \sin t) dt = [\frac{\cos^3 t}{3}]_0^{2\pi} = 0$
	    *   $\int_0^{2\pi} (-\frac{1}{2}\cos t \sin^2 t) dt = [-\frac{\sin^3 t}{6}]_0^{2\pi} = 0$
	    *   $\int_0^{2\pi} (-\frac{1}{2}\sin^3 t) dt = -\frac{1}{2}\int_0^{2\pi} (1-\cos^2 t)\sin t dt = 0$ (奇函数在对称区间上的积分为零)
    因此，原积分等价于：
    $I = \int_0^{2\pi} \cos^2 t dt = \int_0^{2\pi} \frac{1+\cos(2t)}{2} dt$
    $= [\frac{t}{2} + \frac{\sin(2t)}{4}]_0^{2\pi} = \pi$
- 解法 3: [[斯托克斯公式]]  
1. $\oint_L Pdx+Qdy+Rdz = \iint_S (\nabla \times \vec{F}) \cdot d\vec{S}$。
    其中 $\vec{F} = (P, Q, R)$ 是一个向量场，$S$ 是以 $L$ 为边界的任意一个曲面
2. 确定向量场并计算 #旋度
	1. $P=xz, Q=x, R=\frac{y^2}{2}$
	向量场为 $\vec{F} = (xz, x, \frac{y^2}{2})$。
    计算其旋度 $\nabla \times \vec{F}$:
    $\nabla \times \vec{F} = \begin{vmatrix} \mathbf{i} & \mathbf{j} & \mathbf{k} \\ \frac{\partial}{\partial x} & \frac{\partial}{\partial y} & \frac{\partial}{\partial z} \\ xz & x & \frac{y^2}{2} \end{vmatrix}$
	$= \mathbf{i}(\frac{\partial}{\partial y}(\frac{y^2}{2}) - \frac{\partial}{\partial z}(x)) - \mathbf{j}(\frac{\partial}{\partial x}(\frac{y^2}{2}) - \frac{\partial}{\partial z}(xz)) + \mathbf{k}(\frac{\partial}{\partial x}(x) - \frac{\partial}{\partial y}(xz))$
    $= \mathbf{i}(y-0) - \mathbf{j}(0-x) + \mathbf{k}(1-0) = (y, x, 1)$
3. 选择曲面 S 并确定 #法向量  [[曲面法向量与积分方向的确定]]
	最简单的曲面 $S$ 就是被曲线 $L$ 截出的那部分平面 $z=x+y$。
    曲面方程可以写为 $x+y-z=0$。其法向量为 $\vec{n}' = (1, 1, -1)$。
    根据右手定则，曲线 $L$ 的方向（从z轴正向看为逆时针）要求曲面 $S$ 的法向量有朝上的分量（即z分量为正）。因此，我们应取法向量 $\vec{n} = (-1, -1, 1)$。
    单位法向量为 $\vec{n}^0 = \frac{(-1, -1, 1)}{\sqrt{(-1)^2+(-1)^2+1^2}} = (-\frac{1}{\sqrt{3}}, -\frac{1}{\sqrt{3}}, \frac{1}{\sqrt{3}})$
4. 计算曲面积分
	1.  $\iint_S (\nabla \times \vec{F}) \cdot d\vec{S}$ 可以投影到 $xOy$ 平面上计算。
	    $d\vec{S} = \vec{n}^0 dS$
	2.  $(\nabla \times \vec{F}) \cdot \vec{n}^0 = (y, x, 1) \cdot (-\frac{1}{\sqrt{3}}, -\frac{1}{\sqrt{3}}, \frac{1}{\sqrt{3}}) = \frac{-y-x+1}{\sqrt{3}}$ 
	3. 曲面微元 $dS$ 与其在 $xOy$ 平面上的投影 $dxdy$ 的关系是 $dS = \frac{dxdy}{|\vec{n}^0 \cdot \mathbf{k}|} = \frac{dxdy}{1/\sqrt{3}} = \sqrt{3}dxdy$。
	4. 
	    $I = \iint_S ((\nabla \times \vec{F}) \cdot \vec{n}^0) dS = \iint_D (\frac{1-x-y}{\sqrt{3}}) (\sqrt{3}dxdy)$
	    $= \iint_D (1-x-y)dxdy$
	    其中 $D$ 是 $S$ 在 $xOy$ 平面上的投影，即单位圆盘 $x^2+y^2 \le 1$
	二重积分和解法1中得到的结果完全一样，计算结果为 $\pi$

- 题目解析步骤
1. 第一步：写出原二次曲面方程左端的二次型矩阵 
	1. $A = \begin{pmatrix} 1 & a & 1 \\ a & 3 & 1 \\ 1 & 1 & 1 \end{pmatrix}$ 
2. 第二步：分析变换后的 #标准型 ，确定其秩 
	1. 变换后的方程为 $y_1^2 + 4z_1^2 = 4$ 
		1. 左侧的标准形为 $g(x_1, y_1, z_1) = 0x_1^2 + 1y_1^2 + 4z_1^2$
		2. 标准形中只有两项非零（$y_1^2$ 和 $z_1^2$），所以这个二次型的秩为2 
3. 第三步：利用秩不变量建立方程 
	1. 正交变换不改变二次型矩阵的秩，所以原矩阵 $A$ 的秩也必须等于2。
		即 $\text{rank}(A) = 2$ 
	2. 对于一个 $3 \times 3$ 的方阵，它的秩小于3（即秩为0, 1, 或2）的充要条件是它的**行列式为0**。
		所以，我们得到结论：$\det(A) = 0$ 
4. 第四步：计算行列式并求解
	1. 计算矩阵 $A$ 的行列式：
		$\det(A) = \begin{vmatrix} 1 & a & 1 \\ a & 3 & 1 \\ 1 & 1 & 1 \end{vmatrix}$
		$= 1 \cdot \begin{vmatrix} 3 & 1 \\ 1 & 1 \end{vmatrix} - a \cdot \begin{vmatrix} a & 1 \\ 1 & 1 \end{vmatrix} + 1 \cdot \begin{vmatrix} a & 3 \\ 1 & 1 \end{vmatrix}$
		$= 1(3 \cdot 1 - 1 \cdot 1) - a(a \cdot 1 - 1 \cdot 1) + 1(a \cdot 1 - 3 \cdot 1)$
		$= 1(2) - a(a-1) + (a-3)$
		$= 2 - a^2 + a + a - 3$
		$= -a^2 + 2a - 1$
		$= -(a^2 - 2a + 1)$
		$= -(a-1)^2$
	令行列式为0：
	$-(a-1)^2 = 0$
	解得 $a=1$。

- #求解标准型  
	- 可能给出 $a$ 的值（例如，直接告诉你 $a=1$），然后要求你求出该二次型的标准形 
	-  **解法**：
	    1.  写出 $a=1$ 时的矩阵 $A = \begin{pmatrix} 1 & 1 & 1 \\ 1 & 3 & 1 \\ 1 & 1 & 1 \end{pmatrix}$。
	    2.  计算 $A$ 的特征方程 $|\lambda I - A| = 0$。
	    3.  解出所有特征值 $\lambda_1, \lambda_2, \lambda_3$。
	    4.  标准形就是 $\lambda_1 y_1^2 + \lambda_2 y_2^2 + \lambda_3 y_3^2$。
	    *   *验证*：对于 $a=1$，特征方程为 $\lambda(\lambda-1)(\lambda-4)=0$，解得特征值为 $0, 1, 4$。所以标准形为 $y_2^2 + 4y_3^2$，与题目给出的形式完全一致。
- 求解正交变换矩阵P 
	- 在上一问的基础上，进一步要求你找出实现这一变换的 #正交矩阵 $P$  
		**解法**：
	    1.  对每一个特征值 $\lambda_i$，求解线性方程组 $(A - \lambda_i I)\mathbf{x} = \mathbf{0}$，得到其对应的特征向量。
	    2.  将属于不同特征值的特征向量进行**单位化**。如果一个特征值有多个线性无关的特征向量（即重根），则需要对它们进行[[施密特正交化]] 后再[[单位化]]。

- #判断二次型的正定性 [[正定矩阵的定义，判别方式]]
	题目可能会问，当 $a$ 取何值时，该二次型是正定的。
	*   **解法1（特征值法）**：二次型正定的充要条件是其矩阵的所有特征值都大于0。你需要解出含参数 $a$ 的特征值，并令它们全部大于0，解出 $a$ 的范围。
	*   **解法2（顺序主子式法 - Sylvester's Criterion）**：二次型正定的充要条件是其矩阵 $A$ 的所有顺序主子式都大于0。
	    *   一阶主子式：$D_1 = |1| = 1 > 0$
	    *   二阶主子式：$D_2 = \begin{vmatrix} 1 & a \\ a & 3 \end{vmatrix} = 3 - a^2 > 0 \implies -\sqrt{3} < a < \sqrt{3}$
	    *   三阶主子式：$D_3 = \det(A) = -(a-1)^2 > 0$，这个不等式无解。
	    *   结论：该二次型永远不可能是正定的。

		*   $\mu_Y$是$Y$的期望
		*   $\sigma_X^2$是$X$的方差

		*   $\rho$是$X$和$Y$的相关系数

		*   $Y$的期望$E(Y) = \mu$

		*   $Y$的方差$D(Y) = \sigma^2$
		*   $X$和$Y$的相关系数$\rho = 0$
	2. $\rho=0$，所以我们可以直接得出结论：随机变量$X$与$Y$相互独立

	1. 如果随机变量$X$和$Y$相互独立，那么对于任意函数$g(X)$和$h(Y)$，都有：
		$E[g(X)h(Y)] = E[g(X)] \cdot E[h(Y)]$
	2. 本题中，我们需要计算$E(XY^2)$。我们可以令$g(X)=X$  $h(Y)=Y^2$ 
	3. 由于$X$和$Y$相互独立，那么$X$和$Y^2$也相互独立。因此，我们可以将期望拆开：
		$E(XY^2) = E(X) \cdot E(Y^2)$
3. 第三步：分别计算$E(X)$和$E(Y^2)$
	1. 计算$E(X)$ 
		1. $E(X) = \mu$
	 2. 计算$E(Y^2)$

			 1.     $D(Y) = E(Y^2) - [E(Y)]^2$
			 2.   $E(Y^2) = D(Y) + [E(Y)]^2$ 
			 3. 已知$Y$的方差$D(Y)=\sigma^2$和$Y$的期望$E(Y)=\mu$。代入公式：
			    $E(Y^2) = \sigma^2 + \mu^2$
4. $E(XY^2) = E(X) \cdot E(Y^2) = \mu(\sigma^2 + \mu^2)$

1. 关于复合函数求偏导数 
2. #先代后求  
	1. 第一步：求一阶偏导数 $\frac{\partial z}{\partial x}$ 
		1. 对复合函数 $z = f(xy, yg(x))$ 求关于 $x$ 的偏导数。这里需要使用多元复合函数的链式法则
		2. 记 $f$ 对第一个自变量的偏导为 $f'_1$，对第二个自变量的偏导为 $f'_2$
			1. 设 $u = xy$，$v = yg(x)$
			2. $\frac{\partial z}{\partial x} = \frac{\partial f}{\partial u} \frac{\partial u}{\partial x} + \frac{\partial f}{\partial v} \frac{\partial v}{\partial x}$ 
		3. 计算 $\frac{\partial u}{\partial x}$ 和 $\frac{\partial v}{\partial x}$
			*   $\frac{\partial u}{\partial x} = \frac{\partial}{\partial x}(xy) = y$
			*   $\frac{\partial v}{\partial x} = \frac{\partial}{\partial x}(yg(x)) = yg'(x)$ (因为 $y$ 相对于 $x$ 是常数)
		4. 将它们代入链式法则公式：
			$\frac{\partial z}{\partial x} = f'_1(xy, yg(x)) \cdot y + f'_2(xy, yg(x)) \cdot yg'(x)$
	2. 第二步：应用“先代后求”技巧
		1. 计算 $\frac{\partial^2 z}{\partial x \partial y} = \frac{\partial}{\partial y}\left(\frac{\partial z}{\partial x}\right)$ 在 $(1,1)$ 点的值 
			1. 这里的技巧是，在对 $y$ 求导之前，我们可以先把 $x=1$ 这个值代入到 $\frac{\partial z}{\partial x}$ 的表达式中。这样做是合法的，因为我们最终只关心 $x=1$ 这条线上的函数值如何随 $y$ 变化
		2. 将 $x=1$ 代入 $\frac{\partial z}{\partial x}$ 的表达式 
			1. $\left. \frac{\partial z}{\partial x} \right|_{x=1} = f'_1(1 \cdot y, y \cdot g(1)) \cdot y + f'_2(1 \cdot y, y \cdot g(1)) \cdot y \cdot g'(1)$
		3. 使用题目给出的条件 $g(1)=1$ 和 $g'(1)=0$：
			$\left. \frac{\partial z}{\partial x} \right|_{x=1} = f'_1(y, y \cdot 1) \cdot y + f'_2(y, y \cdot 1) \cdot y \cdot 0$
			$\left. \frac{\partial z}{\partial x} \right|_{x=1} = y f'_1(y, y)$
	3. 第三步：对简化后的一阶偏导求导
		1. $\frac{\partial^2 z}{\partial x \partial y} \bigg|_{x=1} = \frac{\partial}{\partial y} \left( \left. \frac{\partial z}{\partial x} \right|_{x=1} \right) = \frac{\partial}{\partial y} [y f'_1(y, y)]$ 
			1. $\frac{\partial}{\partial y} [y f'_1(y, y)] = \frac{\partial(y)}{\partial y} \cdot f'_1(y, y) + y \cdot \frac{\partial}{\partial y}[f'_1(y, y)]$ #乘法法则 和  #链式法则
				$= 1 \cdot f'_1(y, y) + y \cdot \frac{d}{dy}[f'_1(y, y)]$
		2. 处理 $\frac{d}{dy}[f'_1(y, y)]$。这是一个内函数为 $u=y, v=y$ 的 #复合函数求导
			1. 令 $h(y) = f'_1(y, y)$，根据 #全微分的链式法则 ：
					$\frac{d h}{d y} = \frac{\partial f'_1}{\partial u} \frac{du}{dy} + \frac{\partial f'_1}{\partial v} \frac{dv}{dy}$
				$u=y, v=y$，所以 $\frac{du}{dy}=1, \frac{dv}{dy}=1$。
				$\frac{\partial f'_1}{\partial u}$ 是 $f'_1$ 对第一个变量求导，记为 $f''_{11}$。
				$\frac{\partial f'_1}{\partial v}$ 是 $f'_1$ 对第二个变量求导，记为 $f''_{12}$。
				所以，$\frac{d}{dy}[f'_1(y, y)] = f''_{11}(y, y) \cdot 1 + f''_{12}(y, y) \cdot 1 = f''_{11}(y, y) + f''_{12}(y, y)$
		将这个结果代回：
			$\frac{\partial^2 z}{\partial x \partial y} \bigg|_{x=1} = f'_1(y, y) + y [f''_{11}(y, y) + f''_{12}(y, y)]$
	4. 第四步：代入 $y=1$ 求最终值
	5. 将 $y=1$ 代入上式，得到最终答案：
$\left. \frac{\partial^2 z}{\partial x \partial y} \right|_{\substack{x=1 \\ y=1}} = f'_1(1, 1) + 1 \cdot [f''_{11}(1, 1) + f''_{12}(1, 1)]$
$= f'_1(1, 1) + f''_{11}(1, 1) + f''_{12}(1, 1)$

- [[特殊三角函数的图像]] 
- 这道题旨在考察利用导数研究函数性质，进而判断方程根的个数的能力。题目中给出了两种经典的解法： #构造函数法 和 #参数分离法  #参数分离   
- 解法一：构造函数法 
	- 构造一个函数 $f(x)$，将原方程的根转化为函数的零点。通过研究函数的单调性、极值和极限，来判断零点的个数
1. 步骤 1: 构造函数并分析基本性质
	1. 令 $f(x) = k \arctan x - x$。
	该函数的定义域为 $(-\infty, +\infty)$。
	首先，我们检查 $x=0$ 是否为根。$f(0) = k \arctan 0 - 0 = 0$，所以 **$x=0$ 永远是方程的一个根
	2. 分析函数的 #奇偶性
		1. $f(-x) = k \arctan(-x) - (-x) = -k \arctan x + x = -(k \arctan x - x) = -f(x)$
		2. $f(x)$ 是一个**奇函数**。奇函数的图像关于原点对称，这意味着如果 $\xi \neq 0$ 是一个根，那么 $-\xi$ 也必定是一个根。因此，我们只需要研究 $x>0$ 的情况即可
2. 步骤 2: 求导并分析单调性
	1. 对 $f(x)$ 求导：
		$f'(x) = \frac{d}{dx}(k \arctan x - x) = \frac{k}{1+x^2} - 1 = \frac{k - (1+x^2)}{1+x^2} = \frac{k-1-x^2}{1+x^2}$
		1. 导数的符号取决于其分子 $k-1-x^2$ 的符号 
		2. 对参数 $k$ 进行分类讨论。讨论的分界点是使分子性质发生改变的值，即 $k-1=0 \implies k=1$ 
3. 步骤 3: 分类讨论 
	1. **情况一：$k-1 \le 0$，即 $k \le 1$
		1. $k-1 \le 0$。对于任意 $x \neq 0$，$x^2 > 0$，所以分子 $k-1-x^2 < 0$ 
		2. $f'(x) < 0$ 对所有 $x \neq 0$ 成立
			1. 函数 $f(x)$ 在整个定义域 $(-\infty, +\infty)$ 上是严格单调递减的。
			一个严格单调的函数最多只有一个零点。我们已经知道 $f(0)=0$，所以 $x=0$ 是唯一的实根
	2. 情况二：$k-1 > 0$，即 $k > 1$ 
		1. $x>0$ 
			1. 令 $f'(x) = 0$，即 $k-1-x^2 = 0$，解得 $x = \sqrt{k-1}$ (因为我们只考虑 $x>0$)
			2.   当 $0 < x < \sqrt{k-1}$ 时，$x^2 < k-1$，所以 $k-1-x^2 > 0$，$f'(x) > 0$。函数 $f(x)$ 在此区间单调递增
			3. 当 $x > \sqrt{k-1}$ 时，$x^2 > k-1$，所以 $k-1-x^2 < 0$，$f'(x) < 0$。函数 $f(x)$ 在此区间单调递减 
		2. 说明 $f(x)$ 在 $x=\sqrt{k-1}$ 处取得一个极大值 
			1. 因为 $f(x)$ 在 $(0, \sqrt{k-1})$ 上单调递增，且 $f(0)=0$，所以极大值 $f(\sqrt{k-1}) > f(0) = 0$ 
			2. 考察 $x \to +\infty$ 时函数的行为：
			$\lim_{x \to +\infty} f(x) = \lim_{x \to +\infty} (k \arctan x - x)$
			由于 $\arctan x \to \frac{\pi}{2}$ 而 $-x \to -\infty$，所以 $\lim_{x \to +\infty} f(x) = -\infty$
	3. #零点存在性定理  [[零点存在性定理(介值定理)]]  
		1. 在一个连续函数区间 $[a, b]$ 上，如果 $f(a)$ 和 $f(b)$ 异号，则在 $(a, b)$ 内至少存在一个零点。
		在我们这个问题中，函数 $f(x)$ 在 $x=\sqrt{k-1}$ 处取得正的极大值，并且当 $x \to +\infty$ 时趋于负无穷。因此，在区间 $(\sqrt{k-1}, +\infty)$ 内必然存在一个实根，记为 $\xi$
- 总结一下 $k>1$ 的情况： 
	1.  有一个根 $x=0$。
	2.  在 $(0, +\infty)$ 上有一个正根 $\xi$。
	3.  由于 $f(x)$ 是奇函数，在 $(-\infty, 0)$ 上必然还有一个负根 $-\xi$
- 解法二：参数分离法
	1. $k = \frac{x}{\arctan x}$ 
		1. (注意：$x=0$ 是原方程的根，但在分离参数时被排除了，需要最后单独考虑。)
	2. 步骤 2: 构造新函数并分析其性质
		1. $g(x) = \frac{x}{\arctan x}$ 
			1. 求函数 $y=g(x)$ 的图像与水平直线 $y=k$ 的交点个数 
		2. $g(x)$ 的奇偶性：
			$g(-x) = \frac{-x}{\arctan(-x)} = \frac{-x}{-\arctan x} = \frac{x}{\arctan x} = g(x)$
		3. $g(x)$ 是一个**偶函数**，其图像关于 y 轴对称。我们同样只需要研究 $x>0$ 的情况
	3. 步骤 3: 求导分析单调性
		1. $g'(x) = \frac{(x)' \cdot \arctan x - x \cdot (\arctan x)'}{(\arctan x)^2} = \frac{1 \cdot \arctan x - x \cdot \frac{1}{1+x^2}}{(\arctan x)^2} = \frac{\arctan x - \frac{x}{1+x^2}}{(\arctan x)^2}$ 
		2. 判断 $g'(x)$ 的符号，我们只需判断其分子 $\arctan x - \frac{x}{1+x^2}$ 的符号
		3. $h(x) = \arctan x - \frac{x}{1+x^2}$
			1. $h'(x) = \frac{1}{1+x^2} - \frac{1 \cdot (1+x^2) - x \cdot (2x)}{(1+x^2)^2} = \frac{1}{1+x^2} - \frac{1-x^2}{(1+x^2)^2} = \frac{(1+x^2) - (1-x^2)}{(1+x^2)^2} = \frac{2x^2}{(1+x^2)^2}$ 
			2. $x>0$ 时，$h'(x) > 0$  
			3. $h(x)$ 在 $(0, +\infty)$ 上是严格单调递增的
		4. 对于任意 $x>0$，$h(x) > h(0) = \arctan 0 - \frac{0}{1+0} = 0$ 
			1.  $g'(x)$ 的分子为正，所以 $g'(x) > 0$ 对所有 $x>0$ 成立。
			结论：函数 $g(x)$ 在 $(0, +\infty)$ 上是严格单调递增的
	4. 步骤 4: 分析函数图像的边界行为（极限） 
		1.  $g(x)$ 在 $x \to 0^+$ 和 $x \to +\infty$ 时的值，以确定其值域 
			*   $x \to 0^+$ 时的极限：
			    $\lim_{x \to 0^+} g(x) = \lim_{x \to 0^+} \frac{x}{\arctan x}$
			    这是一个 "0/0" 型的极限，可以使用 #洛必达法则 ：
			    $\lim_{x \to 0^+} \frac{(x)'}{(\arctan x)'} = \lim_{x \to 0^+} \frac{1}{1/(1+x^2)} = \lim_{x \to 0^+} (1+x^2) = 1$
			*   $x \to +\infty$ 时的极限：
			    $\lim_{x \to +\infty} g(x) = \lim_{x \to +\infty} \frac{x}{\arctan x}$
			    当 $x \to +\infty$ 时，分子 $x \to +\infty$，分母 $\arctan x \to \frac{\pi}{2}$。 [[特殊三角函数的值]] 
			    所以，$\lim_{x \to +\infty} g(x) = +\infty$。
	5. 步骤 5: 数形结合，判断交点个数
		1. 我们知道 $g(x)$ 在 $(0, +\infty)$ 上从 1 开始单调递增到 $+\infty$。由于 $g(x)$ 是偶函数，其在 $(-\infty, 0)$ 上的图像与右半部分对称 
		2. 判断直线 $y=k$ 与 $y=g(x)$ 的交点个数：
			*   **当 $k \le 1$ 时**：直线 $y=k$ 与 $y=g(x)$ 的图像没有交点。这意味着对于 $x \neq 0$ 没有根。但我们必须回头考虑被排除的 $x=0$。由于 $x=0$ 始终是原方程的根，所以在这种情况下，原方程只有 **一个实根** $x=0$。
			*   **当 $k > 1$ 时**：直线 $y=k$ 与 $y=g(x)$ 在 $(0, +\infty)$ 上有且仅有一个交点（因为 $g(x)$ 在此区间严格单调）。由于 $g(x)$ 是偶函数，它在 $(-\infty, 0)$ 上也必然有且仅有一个交点。这样就有两个非零的根。再加上始终存在的根 $x=0$，总共有 **三个不同的实根**
*   当 $k \le 1$ 时，方程 $k \arctan x - x = 0$ 有 **1** 个实根。
*   当 $k > 1$ 时，方程 $k \arctan x - x = 0$ 有 **3** 个实根。

- #分部积分法 来处理含有高阶偏导数的二重积分，并通过题目给定的边界条件来简化计算过程 
	1. 函数 $f(x,y)$ 具有 #二阶连续偏导数。 这意味着 #混合偏导数相等 ，即 $f''_{xy} = f''_{yx}$，并且所有二阶偏导数都是连续的，这保证了它们在积分区域上是可积的  
	2. $f(1,y)=0$ 和 $f(x,1)=0$。这意味着在积分区域 $D$ 的右边界 ($x=1$) 和上边界 ($y=1$) 上，函数值恒为零 
	3.  积分值：$\iint_D f(x,y) \,dxdy = a$。
	4.  积分区域：$D = \{(x,y) | 0 \le x \le 1, 0 \le y \le 1\}$，是一个单位正方形。
	5.  求解目标：计算二重积分 $I = \iint_D xyf''_{xy}(x,y) \,dxdy$。
1. 第一步：利用边界条件推导其导数信息 
	1. $f(1,y)=0$ 对定义域内所有的 $y$ 成立，所以我们可以将它看作一个关于 $y$ 的函数，这个函数恒为零。对它求关于 $y$ 的偏导数，结果也必然是零 
		1. $f(1,y) = 0 \implies \frac{\partial}{\partial y} f(1,y) = f'_y(1,y) = 0$ 
		2. 同理，因为 $f(x,1)=0$ 对定义域内所有的 $x$ 成立，我们对它求关于 $x$ 的偏导数： 
			1. $f(x,1) = 0 \implies \frac{\partial}{\partial x} f(x,1) = f'_x(x,1) = 0$
2. 第二步：将二重积分化为 #累次积分 并进行第一次 #分部积分法  
	1. 将二重积分 $I$ 转化为累次积分。这里可以选择先对 $y$ 积分，再对 $x$ 积分。
		$I = \iint_D xyf''_{xy}(x,y) \,dxdy = \int_0^1 \left[ \int_0^1 xyf''_{xy}(x,y) \,dy \right] dx$
	2. 计算内层的积分 $\int_0^1 xyf''_{xy}(x,y) \,dy$ 
		1.  $f''_{xy}(x,y) = \frac{\partial}{\partial y} (f'_x(x,y))$，所以内层积分可以写成：
			$\int_0^1 x \cdot y \cdot \frac{\partial}{\partial y} (f'_x(x,y)) \,dy = x \int_0^1 y \,d(f'_x(x,y))$
		2. 单变量积分中 #分部积分法 的形式 $\int u \,dv = uv - \int v \,du$。
			令 $u=y$, $dv = d(f'_x(x,y))$。则 $du=dy$, $v=f'_x(x,y)$
		3. 所以：
			$x \int_0^1 y \,d(f'_x(x,y)) = x \left( \left[ y f'_x(x,y) \right]_{y=0}^{y=1} - \int_0^1 f'_x(x,y) \,dy \right)$
			$= x \left( (1 \cdot f'_x(x,1) - 0 \cdot f'_x(x,0)) - \int_0^1 f'_x(x,y) \,dy \right)$
	根据**第一步**的结论，我们知道 $f'_x(x,1)=0$。代入上式：
	$= x \left( 0 - \int_0^1 f'_x(x,y) \,dy \right) = -x \int_0^1 f'_x(x,y) \,dy$
3. 第三步：代入外层积分并进行第二次分部积分 
	1. $I = \int_0^1 \left( -x \int_0^1 f'_x(x,y) \,dy \right) dx = - \int_0^1 \int_0^1 x f'_x(x,y) \,dy \,dx$ 
	2. 由于被积函数连续，根据富比尼定理，我们可以交换积分次序： [[二重积分的积分次序交换]]  
		$I = - \int_0^1 \left[ \int_0^1 x f'_x(x,y) \,dx \right] dy$
	3. 处理内层的积分 $\int_0^1 x f'_x(x,y) \,dx$
		1. 在这一步中，$y$ 被视为常数。
			注意到 $f'_x(x,y) = \frac{\partial}{\partial x} (f(x,y))$，所以内层积分可以写成：
			$\int_0^1 x \,d(f(x,y))$
			1. #分部积分法  令 $u=x$, $dv = d(f(x,y))$。则 $du=dx$, $v=f(x,y)$
				1. $\int_0^1 x \,d(f(x,y)) = \left[ x f(x,y) \right]_{x=0}^{x=1} - \int_0^1 f(x,y) \,dx$ 
				   $= (1 \cdot f(1,y) - 0 \cdot f(0,y)) - \int_0^1 f(x,y) \,dx$
				2. 给定的原始条件，$f(1,y)=0$ 
					1. $= 0 - \int_0^1 f(x,y) \,dx = - \int_0^1 f(x,y) \,dx$
4. 第四步：得出最终结果 
	1. 上一步的结果代入 $I$ 的表达式
	2. $I = - \int_0^1 \left( - \int_0^1 f(x,y) \,dx \right) dy$
		$I = \int_0^1 \left( \int_0^1 f(x,y) \,dx \right) dy = \int_0^1 \int_0^1 f(x,y) \,dx \,dy$
	正是 $\iint_D f(x,y) \,dxdy$ 的累次积分形式。根据题目已知条件，该积分的值为 $a$。
	因此，$I=a$

 (I) 求解 #特征值与特征向量
 1. [[矩阵乘法的列向量解释]]  
	 1. 矩阵方程 $A[v_1, v_2, ..., v_k] = [w_1, w_2, ..., w_k]$ 等价于一系列独立的向量方程 $Av_1=w_1, Av_2=w_2, ..., Av_k=w_k$ 
 2. [[特征值和特征向量的定义]] 
	 1.  若 $Ax = \lambda x$ (其中 $x$ 为非零向量)，则 $\lambda$ 是特征值，$x$ 是对应的特征向量
3. [[秩与特征值0的关系]]  
	1. $n$ 阶方阵 $A$ 的秩 $r(A) < n$ 的充要条件是 $\lambda=0$ 是 $A$ 的一个特征值  
4. [[实对称矩阵的性质]]  
	1.  不同特征值对应的特征向量相互正交 
- 推导步骤 
1. 第一步：分解矩阵方程，寻找特征值和特征向量 
	1. 给定的矩阵方程 $A\begin{pmatrix} 1 & 1 \\ 0 & 0 \\ -1 & 1 \end{pmatrix} = \begin{pmatrix} -1 & 1 \\ 0 & 0 \\ 1 & 1 \end{pmatrix}$ 可以分解为两个独立的向量方程 
		1.  $A\begin{pmatrix} 1 \\ 0 \\ -1 \end{pmatrix} = \begin{pmatrix} -1 \\ 0 \\ 1 \end{pmatrix}$
		2.  $A\begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix} = \begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix}$
	2. 对于方程1，我们可以发现右边的向量是左边向量的 $-1$ 倍：
	    $\begin{pmatrix} -1 \\ 0 \\ 1 \end{pmatrix} = -1 \cdot \begin{pmatrix} 1 \\ 0 \\ -1 \end{pmatrix}$
		该方程可以写成 $A\begin{pmatrix} 1 \\ 0 \\ -1 \end{pmatrix} = -1 \cdot \begin{pmatrix} 1 \\ 0 \\ -1 \end{pmatrix}$ 
		根据特征值的定义，我们找到了第一个特征值 $\lambda_1 = -1$，以及它对应的一个特征向量 $\alpha_1 = \begin{pmatrix} 1 \\ 0 \\ -1 \end{pmatrix}$
	3. 对于方程2，我们可以发现右边的向量与左边的向量完全相同：
	    $\begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix} = 1 \cdot \begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix}$ 
	    该方程可以写成 $A\begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix} = 1 \cdot \begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix}$ 
	    根据特征值的定义，我们找到了第二个特征值 $\lambda_2 = 1$，以及它对应的一个特征向量 $\alpha_2 = \begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix}$。 
2. 第二步：利用秩的信息找到第三个特征值 
	1. 题目给出 $A$ 是3阶矩阵，且秩 $r(A)=2$ 
	2. 因为 $r(A) = 2 < 3$ (矩阵的阶数)，所以矩阵 $A$ 是奇异矩阵，其行列式 $|A|=0$
	3. 对于齐次线性方程组 $Ax=0$，因为 $r(A) < 3$，该方程必有非零解  
		1. 将方程写成 $Ax=0x$ 的形式，根据特征值的定义，我们确定 $\lambda_3 = 0$ 是 $A$ 的第三个特征值 
3. 第三步：求解第三个特征值对应的特征向量  
	1. 已经找到了三个不同的特征值 $\lambda_1=-1, \lambda_2=1, \lambda_3=0$ 
	2. 设 $\lambda_3=0$ 对应的特征向量为 $\alpha_3 = (x, y, z)^T$
	3. 由于 $A$ 是 #实对称矩阵 ，不同特征值对应的 #特征向量相互正交 。因此，$\alpha_3$ 必须同时与 $\alpha_1$ 和 $\alpha_2$ 正交 
		1.  $\alpha_3$ 与 $\alpha_1$ 正交: $\alpha_1^T \alpha_3 = 0$
		    $(1, 0, -1) \begin{pmatrix} x \\ y \\ z \end{pmatrix} = x - z = 0 \implies x=z$
		2.  $\alpha_3$ 与 $\alpha_2$ 正交: $\alpha_2^T \alpha_3 = 0$
		    $(1, 0, 1) \begin{pmatrix} x \\ y \\ z \end{pmatrix} = x + z = 0$
4. 联立方程组：
	$\begin{cases} x - z = 0 \\ x + z = 0 \end{cases}$
	解得 $x=0, z=0$。对于 $y$ 没有限制，所以 $y$ 可以是任意非零实数。

5. 结论 (I)

	*   特征值 $\lambda_2 = 1$，对应的全部特征向量为 $k_2 \begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix}$ ($k_2 \neq 0$)。

- (II) 求解矩阵 A 
	- #谱定理    [[实对称矩阵的性质]]  
		- 任何实对称矩阵 $A$ 都可以被正交对角化，即存在一个正交矩阵 $Q$ 和一个对角矩阵 $\Lambda$，使得 $A = Q \Lambda Q^T$
			*   $\Lambda$ 是由 $A$ 的特征值构成的对角矩阵。
		    *   $Q$ 的列向量是由 $A$ 对应的特征向量经过**单位化**后构成的标准正交基。
		    *   正交矩阵满足 $Q^{-1} = Q^T$。
1. 第一步：构建对角矩阵 $\Lambda$ 
	1. 将特征值排列在对角线上（顺序必须与后面 $Q$ 矩阵中特征向量的顺序一一对应）。我们按 $\lambda_1, \lambda_2, \lambda_3$ 的顺序排列
		1. $\Lambda = \begin{pmatrix} \lambda_1 & 0 & 0 \\ 0 & \lambda_2 & 0 \\ 0 & 0 & \lambda_3 \end{pmatrix} = \begin{pmatrix} -1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 0 \end{pmatrix}$ 
2. 第二步：构建正交矩阵 $Q$
	需要将特征向量 $\alpha_1, \alpha_2, \alpha_3$ 单位化。
	*   $\alpha_1 = \begin{pmatrix} 1 \\ 0 \\ -1 \end{pmatrix}$, 模长 $||\alpha_1|| = \sqrt{1^2+0^2+(-1)^2} = \sqrt{2}$。
	    单位化后 $q_1 = \frac{\alpha_1}{||\alpha_1||} = \begin{pmatrix} 1/\sqrt{2} \\ 0 \\ -1/\sqrt{2} \end{pmatrix}$。
	*   $\alpha_2 = \begin{pmatrix} 1 \\ 0 \\ 1 \end{pmatrix}$, 模长 $||\alpha_2|| = \sqrt{1^2+0^2+1^2} = \sqrt{2}$。
	    单位化后 $q_2 = \frac{\alpha_2}{||\alpha_2||} = \begin{pmatrix} 1/\sqrt{2} \\ 0 \\ 1/\sqrt{2} \end{pmatrix}$。
	*   $\alpha_3 = \begin{pmatrix} 0 \\ 1 \\ 0 \end{pmatrix}$, 模长 $||\alpha_3|| = \sqrt{0^2+1^2+0^2} = 1$。
	    单位化后 $q_3 = \frac{\alpha_3}{||\alpha_3||} = \begin{pmatrix} 0 \\ 1 \\ 0 \end{pmatrix}$。
	1. 将单位化的特征向量作为列向量，构成正交矩阵 $Q$ (顺序与 $\Lambda$ 对应):
	$Q = [q_1, q_2, q_3] = \begin{pmatrix} 1/\sqrt{2} & 1/\sqrt{2} & 0 \\ 0 & 0 & 1 \\ -1/\sqrt{2} & 1/\sqrt{2} & 0 \end{pmatrix}$
3. 第三步：计算 $A = Q \Lambda Q^T$ 
	1. $Q^T = \begin{pmatrix} 1/\sqrt{2} & 0 & -1/\sqrt{2} \\ 1/\sqrt{2} & 0 & 1/\sqrt{2} \\ 0 & 1 & 0 \end{pmatrix}$ 
		1. $A = Q \Lambda Q^T = \begin{pmatrix} 1/\sqrt{2} & 1/\sqrt{2} & 0 \\ 0 & 0 & 1 \\ -1/\sqrt{2} & 1/\sqrt{2} & 0 \end{pmatrix} \begin{pmatrix} -1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 0 \end{pmatrix} \begin{pmatrix} 1/\sqrt{2} & 0 & -1/\sqrt{2} \\ 1/\sqrt{2} & 0 & 1/\sqrt{2} \\ 0 & 1 & 0 \end{pmatrix}$
4. 先计算 $Q\Lambda$:
	$Q\Lambda = \begin{pmatrix} -1/\sqrt{2} & 1/\sqrt{2} & 0 \\ 0 & 0 & 0 \\ 1/\sqrt{2} & 1/\sqrt{2} & 0 \end{pmatrix}$
	再计算 $(Q\Lambda)Q^T$:
	$A = \begin{pmatrix} -1/\sqrt{2} & 1/\sqrt{2} & 0 \\ 0 & 0 & 0 \\ 1/\sqrt{2} & 1/\sqrt{2} & 0 \end{pmatrix} \begin{pmatrix} 1/\sqrt{2} & 0 & -1/\sqrt{2} \\ 1/\sqrt{2} & 0 & 1/\sqrt{2} \\ 0 & 1 & 0 \end{pmatrix}$
	$A = \begin{pmatrix} (-1/2)+(1/2)+0 & 0 & (1/2)+(1/2)+0 \\ 0 & 0 & 0 \\ (1/2)+(1/2)+0 & 0 & (-1/2)+(1/2)+0 \end{pmatrix} = \begin{pmatrix} 0 & 0 & 1 \\ 0 & 0 & 0 \\ 1 & 0 & 0 \end{pmatrix}$

**结论 (II):**
矩阵 $A = \begin{pmatrix} 0 & 0 & 1 \\ 0 & 0 & 0 \\ 1 & 0 & 0 \end{pmatrix}$

		*   **$E(X)$**: 根据 $X$ 的边缘分布计算。
	        $E(X) = \sum x_i P(X=x_i) = 0 \times \frac{1}{3} + 1 \times \frac{2}{3} = \frac{2}{3}$
	    *   **$E(Y)$**: 根据 $Y$ 的边缘分布计算。
	        $E(Y) = \sum y_j P(Y=y_j) = (-1) \times \frac{1}{3} + 0 \times \frac{1}{3} + 1 \times \frac{1}{3} = 0$
	    *   **$E(XY)$**: 这就是 $E(Z)$。根据第二问中 $Z$ 的分布计算。
		   $E(XY) = E(Z) = \sum z_k P(Z=z_k) = (-1) \times \frac{1}{3} + 0 \times \frac{1}{3} + 1 \times \frac{1}{3} = 0$
	3. 计算协方差 $Cov(X,Y)$
		1. $Cov(X,Y) = E(XY) - E(X)E(Y) = 0 - (\frac{2}{3}) \times 0 = 0$
	4. 计算相关系数 $\rho_{XY}$ 
		由于协方差 $Cov(X,Y)=0$，只要 $X$ 和 $Y$ 的方差不为零，相关系数就为零。
		$\rho_{XY} = \frac{Cov(X,Y)}{\sqrt{DX \cdot DY}} = \frac{0}{\sqrt{DX \cdot DY}} = 0$
		（为了严谨，我们可以计算一下方差确保分母不为0）
		*   **$DX$**:
			$E(X^2) = 0^2 \times \frac{1}{3} + 1^2 \times \frac{2}{3} = \frac{2}{3}$
			$DX = E(X^2) - [E(X)]^2 = \frac{2}{3} - (\frac{2}{3})^2 = \frac{2}{3} - \frac{4}{9} = \frac{2}{9}$
		*   **$DY$**:
			$E(Y^2) = (-1)^2 \times \frac{1}{3} + 0^2 \times \frac{1}{3} + 1^2 \times \frac{1}{3} = \frac{1}{3} + \frac{1}{3} = \frac{2}{3}$
			$DY = E(Y^2) - [E(Y)]^2 = \frac{2}{3} - 0^2 = \frac{2}{3}$
	因为 $DX > 0$ 且 $DY > 0$，所以分母不为零，$\rho_{XY}=0$ 的结论是正确的

 - 联合分布与边缘分布  [[联合分布与边缘分布]] 
	 - 核心关系 
		 - $P(X=x_i) = \sum_{j} P(X=x_i, Y=y_j)$ （行求和）；$P(Y=y_j) = \sum_{i} P(X=x_i, Y=y_j)$ （列求和
		反过来，如果只知道边缘分布，通常是无法唯一确定联合分布的，除非有额外的条件，就像本题的 $P\{X^2=Y^2\}=1$ 
- 随机变量的独立性 
	-    **定义**：若对任意 $x_i, y_j$ 都有 $P(X=x_i, Y=y_j) = P(X=x_i)P(Y=y_j)$，则称 $X$ 和 $Y$ 相互独立
	- 衍生考点 
		本题是一个绝佳的反例。我们计算出 $\rho_{XY}=0$，说明 $X$ 和 $Y$ **不相关**。但是它们独立吗？我们来检验一下：
		*   $P\{X=1, Y=1\} = \frac{1}{3}$
		*   $P(X=1)P(Y=1) = \frac{2}{3} \times \frac{1}{3} = \frac{2}{9}$
		*   因为 $P\{X=1, Y=1\} \neq P(X=1)P(Y=1)$，所以 $X$ 和 $Y$ **不独立**。
		*   **重要结论**：**随机变量相互独立一定不相关；但不相关不一定相互独立**。这是一个非常重要的考点，本题的结果恰好可以说明这一点。只有在 $(X,Y)$ 服从二维正态分布的特殊情况下，不相关才能推出独立。

 #较复杂  

- 考察了数理统计中的两个核心知识点：[[最大似然估计法]]   [[估计量的性质]]  [[相合估计量]]
1. 问题 (I): 求参数 $\sigma^2$ 的 #最大似然估计 $\hat{\sigma}^2$ 
	1. 核心思想是：寻找一个参数值，使得当前这组样本观测值出现的概率（似然）最大。
 2. 第一步：写出似然函数 $L(\sigma^2)$
	 1. #似然函数 是 #样本联合概率密度函数  。因为 $X_1, X_2, \dots, X_n$ 是来自正态总体 $N(\mu_0, \sigma^2)$ 的简单随机样本，它们相互独立且同分布。
		单个样本 $X_i$ 的概率密度函数 (PDF) 为：
		$f(x_i; \sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left\{-\frac{(x_i-\mu_0)^2}{2\sigma^2}\right\}$
		- 样本是独立同分布的，所以 $n$ 个样本的联合概率密度函数就是各自概率密度函数的乘积：
		$L(\sigma^2) = \prod_{i=1}^{n} f(x_i; \sigma^2) = \prod_{i=1}^{n} \frac{1}{\sqrt{2\pi\sigma^2}} \exp\left\{-\frac{(x_i-\mu_0)^2}{2\sigma^2}\right\}$
		$L(\sigma^2) = \left(\frac{1}{\sqrt{2\pi\sigma^2}}\right)^n \exp\left\{-\sum_{i=1}^{n}\frac{(x_i-\mu_0)^2}{2\sigma^2}\right\}$
		$L(\sigma^2) = (2\pi\sigma^2)^{-n/2} \exp\left\{-\frac{1}{2\sigma^2}\sum_{i=1}^{n}(x_i-\mu_0)^2\right\}$  
3. 第二步：取对数，得到对数似然函数 $\ln L(\sigma^2)$ 
	1. 对 $L(\sigma^2)$ 求导比较复杂，因为有乘积和指数。通常我们通过取对数将其转化为加法，这并不会改变极值点的位置，因为对数函数是单调递增的 
		$\ln L(\sigma^2) = \ln\left[(2\pi\sigma^2)^{-n/2} \exp\left\{-\frac{1}{2\sigma^2}\sum_{i=1}^{n}(x_i-\mu_0)^2\right\}\right]$
		$\ln L(\sigma^2) = -\frac{n}{2}\ln(2\pi\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^{n}(x_i-\mu_0)^2$
4. 第三步：对参数求导，并令其为 0 
	为了找到使 $\ln L(\sigma^2)$ 最大的 $\sigma^2$ 值，我们对 $\sigma^2$ 求导（注意，这里我们将 $\sigma^2$ 视为一个整体变量），并令导数为零。
		$\frac{d[\ln L(\sigma^2)]}{d(\sigma^2)} = \frac{d}{d(\sigma^2)}\left[-\frac{n}{2}\ln(2\pi) - \frac{n}{2}\ln(\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^{n}(x_i-\mu_0)^2\right]$
		$\frac{d[\ln L(\sigma^2)]}{d(\sigma^2)} = 0 - \frac{n}{2} \cdot \frac{1}{\sigma^2} - \left(-\frac{1}{2(\sigma^2)^2}\right)\sum_{i=1}^{n}(x_i-\mu_0)^2 = 0$
		$-\frac{n}{2\sigma^2} + \frac{1}{2(\sigma^2)^2}\sum_{i=1}^{n}(x_i-\mu_0)^2 = 0$
	两边同乘以 $2(\sigma^2)^2$ 可得：
		$-n\sigma^2 + \sum_{i=1}^{n}(x_i-\mu_0)^2 = 0$
5. 第四步：解出参数的估计值
	从上式解出 $\sigma^2$：
	$\sigma^2 = \frac{1}{n}\sum_{i=1}^{n}(x_i-\mu_0)^2$
	这个解就是最大似然估计量，我们用 $\hat{\sigma}^2$ 表示，并把样本观测值 $x_i$ 换回随机变量 $X_i$：
	$\hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^{n}(X_i-\mu_0)^2$
- 问题 (II): 计算 $E(\hat{\sigma}^2)$ 和 $D(\hat{\sigma}^2)$ 
1. 解法 1：利用[[抽样分布的基本性质（样本均值）]]（$\chi^2$ 分布）  #抽样分布理论  
	1. 这是统计学中一个非常重要的定理
		1. 若 $X_1, X_2, \dots, X_n$ 是来自正态总体 $N(\mu, \sigma^2)$ 的简单随机样本，则 $\frac{(X_i-\mu)}{\sigma} \sim N(0, 1)$。并且， $n$ 个独立的标准正态分布变量的平方和服从自由度为 $n$ 的卡方分布 ($\chi^2$ 分布)。
		2. 即：$\sum_{i=1}^{n}\left(\frac{X_i-\mu}{\sigma}\right)^2 \sim \chi^2(n)$  [[抽样分布理论]] 
	2. 本题中，$\mu = \mu_0$ 是已知的，所以：
		$\sum_{i=1}^{n}\left(\frac{X_i-\mu_0}{\sigma}\right)^2 \sim \chi^2(n)$ 
		- 将我们的估计量 $\hat{\sigma}^2$ 与这个式子联系起来：
			$\frac{n\hat{\sigma}^2}{\sigma^2} = \frac{n}{\sigma^2} \cdot \frac{1}{n}\sum_{i=1}^{n}(X_i-\mu_0)^2 = \frac{1}{\sigma^2}\sum_{i=1}^{n}(X_i-\mu_0)^2 = \sum_{i=1}^{n}\left(\frac{X_i-\mu_0}{\sigma}\right)^2$ 
	3. 得到关键关系：
		$\frac{n\hat{\sigma}^2}{\sigma^2} \sim \chi^2(n)$
2.    [[分布的种类]]   #抽样分布  [[抽样分布]] 
	1. 我们利用 $\chi^2(n)$ 分布的期望和方差性质： #卡方分布  [[卡方分布]] 
		若 $Y \sim \chi^2(n)$，则 $E(Y) = n$ 且 $D(Y) = 2n$ 
3. 计算期望 $E(\hat{\sigma}^2)$
	1. $E\left(\frac{n\hat{\sigma}^2}{\sigma^2}\right) = n$
	根据  [[抽样分布的基本性质（样本均值）]]   $E(aX) = aE(X)$，我们有：  [[期望与方差的性质和定义]] 
		$\frac{n}{\sigma^2}E(\hat{\sigma}^2) = n$
		$E(\hat{\sigma}^2) = \sigma^2$
由于估计量的期望等于参数的真值，我们称 $\hat{\sigma}^2$ 是 $\sigma^2$ 的 #无偏估计量 
4. **计算方差 $D(\hat{\sigma}^2)$:**
	$D\left(\frac{n\hat{\sigma}^2}{\sigma^2}\right) = 2n$
	根据方差的性质 $D(aX) = a^2D(X)$，我们有：
	$\left(\frac{n}{\sigma^2}\right)^2 D(\hat{\sigma}^2) = 2n$
	$\frac{n^2}{\sigma^4}D(\hat{\sigma}^2) = 2n$
	$D(\hat{\sigma}^2) = \frac{2n\sigma^4}{n^2} = \frac{2\sigma^4}{n}$ 
- 解法 2：利用 #期望和方差的定义   
	
1. **计算期望 $E(\hat{\sigma}^2)$:**
	$E(\hat{\sigma}^2) = E\left[\frac{1}{n}\sum_{i=1}^{n}(X_i-\mu_0)^2\right]$
2. 利用 #期望的线性性质 ：
	$E(\hat{\sigma}^2) = \frac{1}{n}\sum_{i=1}^{n}E[(X_i-\mu_0)^2]$
	1. 根据 #方差的定义 ，[[期望与方差的性质和定义]] $D(X_i) = E[(X_i-E(X_i))^2]$。因为 $E(X_i) = \mu_0$，所以 $D(X_i) = E[(X_i-\mu_0)^2]$。 
		又因为 $X_i$ 来自 $N(\mu_0, \sigma^2)$，所以 $D(X_i) = \sigma^2$。
		因此，$E[(X_i-\mu_0)^2] = \sigma^2$。
	$E(\hat{\sigma}^2) = \frac{1}{n}\sum_{i=1}^{n}\sigma^2 = \frac{1}{n}(n\sigma^2) = \sigma^2$
3. 计算方差 $D(\hat{\sigma}^2)$: 
	$D(\hat{\sigma}^2) = D\left[\frac{1}{n}\sum_{i=1}^{n}(X_i-\mu_0)^2\right]$
	1. 因为样本 $X_i$ 相互独立，所以 $(X_i-\mu_0)^2$ 也相互独立。根据 #方差性质 $D(\sum a_i X_i) = \sum a_i^2 D(X_i)$（当 $X_i$ 独立时）：
		$D(\hat{\sigma}^2) = \frac{1}{n^2}\sum_{i=1}^{n}D[(X_i-\mu_0)^2]$
	2. 由于 $X_i$ 同分布，所以 $D[(X_i-\mu_0)^2]$ 对所有 $i$ 都相同。
		$D(\hat{\sigma}^2) = \frac{1}{n^2} \cdot n \cdot D[(X_1-\mu_0)^2] = \frac{1}{n}D[(X_1-\mu_0)^2]$
	3. 我们利用公式 $D(Y) = E(Y^2) - [E(Y)]^2$，令 $Y = (X_1-\mu_0)^2$。
		$D[(X_1-\mu_0)^2] = E\left[((X_1-\mu_0)^2)^2\right] - \left(E[(X_1-\mu_0)^2]\right)^2$
		$D[(X_1-\mu_0)^2] = E[(X_1-\mu_0)^4] - (\sigma^2)^2$
	4. 为了计算 $E[(X_1-\mu_0)^4]$，我们先将变量标准化。令 $Z = \frac{X_1-\mu_0}{\sigma} \sim N(0, 1)$。
		$E[(X_1-\mu_0)^4] = E[(\sigma Z)^4] = \sigma^4 E[Z^4]$
		对于标准正态分布 $Z \sim N(0, 1)$，其 #二阶矩 $E[Z^2]=1$， #四阶矩 $E[Z^4]=3$。 [[一阶矩]] 
		所以，$E[(X_1-\mu_0)^4] = 3\sigma^4$。
		代入得：
	$D[(X_1-\mu_0)^2] = 3\sigma^4 - (\sigma^2)^2 = 2\sigma^4$
	最后，我们得到 $D(\hat{\sigma}^2)$：
	$D(\hat{\sigma}^2) = \frac{1}{n}(2\sigma^4) = \frac{2\sigma^4}{n}$

- 衍生考点
1. $\mu$ 未知的情况 这是最常见的变种。如果题目条件变为 $X_i \sim N(\mu, \sigma^2)$，其中 $\mu$ 和 $\sigma^2$ 均未知。
    *   $\mu$ 的最大似然估计是样本均值 $\hat{\mu} = \bar{X} = \frac{1}{n}\sum X_i$。
    *   $\sigma^2$ 的最大似然估计变为 $\hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^{n}(X_i-\bar{X})^2$。
    *   此时，关键的抽样分布变为 $\frac{(n-1)S^2}{\sigma^2} = \frac{\sum(X_i-\bar{X})^2}{\sigma^2} \sim \chi^2(n-1)$，自由度变为 $n-1$（因为估计 $\mu$ 用掉了一个自由度）。
    *   在这种情况下，$E(\hat{\sigma}^2) = E\left(\frac{1}{n}\sum(X_i-\bar{X})^2\right) = \frac{n-1}{n}\sigma^2$。这是一个 #有偏估计 
2. #无偏估计量  
	1.   引出样本方差 $S^2 = \frac{1}{n-1}\sum_{i=1}^{n}(X_i-\bar{X})^2$。可以证明 $E(S^2) = \sigma^2$，所以 $S^2$ 是 $\sigma^2$ 的无偏估计量。考试常会要求比较 MLE 和无偏估计量。

4. 区间估计 
	*   利用 $\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$ 这个抽样分布，可以构造参数 $\sigma^2$ 的 #置信区间 。
5. 假设检验
	  *   同样利用 $\chi^2$ 抽样分布，可以对关于 $\sigma^2$ 的假设（如 $H_0: \sigma^2 = \sigma_0^2$）进行检验
