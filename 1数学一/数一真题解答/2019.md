[[可微、连续、偏导数存在的关系]]  
 函数 $f(x)$ 在点 $x=a$ 处可导的**充要条件**是左导数和右导数都存在且相等。即 $f'_{-}(a) = f'_{+}(a)$。
1. #导数的定义   
	1. $f'_{-}(0) = \lim_{x \to 0^-} \frac{f(x) - f(0)}{x - 0} = \lim_{x \to 0^-} \frac{-x^2 - 0}{x} = \lim_{x \to 0^-} (-x) = 0$
	    所以，左导数存在且等于 $0$
	2. $f'_{+}(0) = \lim_{x \to 0^+} \frac{f(x) - f(0)}{x - 0} = \lim_{x \to 0^+} \frac{x\ln x - 0}{x} = \lim_{x \to 0^+} \ln x$
	    当 $x$ 从正方向无限趋近于 $0$ 时，$\ln x$ 趋近于 $-\infty$。所以，这个极限不存在
	3. 左、右导数不相等。因此，函数 $f(x)$ 在 $x=0$ 处**不可导**
2. #判断是否为极值点  
	*   左正右负（$f'(x)$ 在 $x_0$ 左侧 $>0$，右侧 $<0$），则 $x_0$ 为极大值点。
	*   左负右正（$f'(x)$ 在 $x_0$ 左侧 $<0$，右侧 $>0$），则 $x_0$ 为极小值点。
	**重要前提**：使用此方法前，需要先确认函数在 $x=0$ 处是否连续。
	*   左极限：$\lim_{x \to 0^-} f(x) = \lim_{x \to 0^-} (-x^2) = 0$。
	*   右极限：$\lim_{x \to 0^+} f(x) = \lim_{x \to 0^+} (x\ln x)$。这是一个 $0 \cdot \infty$ 型的未定式，可以通过洛必达法则计算 $\lim_{x \to 0^+} \frac{\ln x}{1/x} = \lim_{x \to 0^+} \frac{1/x}{-1/x^2} = \lim_{x \to 0^+} (-x) = 0$。
	*   函数值：$f(0) = 0$。
	因为左极限 = 右极限 = 函数值，所以函数在 $x=0$ 处是**连续的**。现在我们可以使用第一导数判别法
	1.  **分析 $x=0$ 左侧的导数符号**：
	    当 $x < 0$ 时，$f(x) = -x^2$，其导数为 $f'(x) = -2x$。
	    因为 $x < 0$，所以 $-2x > 0$。
	    这意味着在 $x=0$ 的左侧邻域内，函数是**单调递增**的。   
	2. **分析 $x=0$ 右侧的导数符号**：
	    当 $x > 0$ 时，$f(x) = x\ln x$，其导数为 $f'(x) = (x)'\ln x + x(\ln x)' = \ln x + 1$。
	    我们需要判断在 $x=0$ 的一个很小的右邻域内 $f'(x)$ 的符号。当 $x \to 0^+$ 时，$\ln x \to -\infty$，所以 $\ln x + 1$ 必然是负数。
	    更精确地，令 $f'(x) = \ln x + 1 = 0$，解得 $x = e^{-1}$。
	    在区间 $(0, e^{-1})$ 内，$\ln x < -1$，所以 $f'(x) = \ln x + 1 < 0$。
	    这意味着在 $x=0$ 的右侧邻域（例如 $(0, e^{-1})$）内，函数是**单调递减**的。 
	3. 函数 $f(x)$ 在 $x=0$ 的左侧单调递增 ($f'(x)>0$)，在右侧单调递减 ($f'(x)<0$)。根据第一导数判别法，函数在 $x=0$ 处取得**极大值**。
	    因此，$x=0$ 是一个**极值点**。

[[级数收敛的必要条件]] 
如果级数 $\sum u_n$ 收敛，那么它的一般项 $u_n$ 必须趋向于0，即 $\lim_{n \to \infty} u_n = 0$。
[[几何级数的求和公式]]
1. (D) $\sum_{n=1}^{\infty} (u_{n+1}^2 - u_n^2)$ 
	1. 典型的**伸缩级数**。我们来考察它的部分和 $S_n$： #伸缩级数 [[伸缩级数]]
		$S_n = \sum_{k=1}^{n} (u_{k+1}^2 - u_k^2)$
		展开后得到：
		$S_n = (u_2^2 - u_1^2) + (u_3^2 - u_2^2) + (u_4^2 - u_3^2) + \dots + (u_{n+1}^2 - u_n^2)$
	2. $\lim_{n \to \infty} S_n = \lim_{n \to \infty} (u_{n+1}^2 - u_1^2)$ 
	3. 已经知道 $\lim_{n \to \infty} u_n = a$，所以 $\lim_{n \to \infty} u_{n+1} = a$
		1. $\lim_{n \to \infty} S_n = (\lim_{n \to \infty} u_{n+1})^2 - u_1^2 = a^2 - u_1^2$ 
	4. 因为 $a$ 和 $u_1$ 都是有限值，所以极限 $a^2 - u_1^2$ 是一个有限值。根据级数收敛的定义，该级数收敛。
		**所以选项 (D) 是正确的
2. (A) $\sum_{n=1}^{\infty} \frac{u_n}{n}$  #举反例 
	1.  $u_n = -\frac{1}{n}$级数变为 $\sum_{n=1}^{\infty} (-1)^n \frac{1}{-1/n} = \sum_{n=1}^{\infty} (-1)^n (-n) = \sum_{n=1}^{\infty} (-1)^{n+1} n$。
    这个级数的通项为 $a_n = (-1)^{n+1} n$。显然 $\lim_{n \to \infty} a_n \neq 0$。根据级数收敛的必要条件，该级数发散
3. (C) $\sum_{n=1}^{\infty} (1 - \frac{u_n}{u_{n+1}})$  
	1.  $u_n = -\frac{1}{n}$ 
	2. 级数变为 $\sum_{n=1}^{\infty} (-1)^n \frac{1}{-1/n} = \sum_{n=1}^{\infty} (-1)^n (-n) = \sum_{n=1}^{\infty} (-1)^{n+1} n$。
	    这个级数的通项为 $a_n = (-1)^{n+1} n$。显然 $\lim_{n \to \infty} a_n \neq 0$。根据级数收敛的必要条件，该级数发散。
4. (B) $\sum_{n=1}^{\infty} (-1)^n \frac{1}{u_n}$ 
	1.  $u_n = -\frac{1}{n}$级数变为 $\sum_{n=1}^{\infty} (-1)^n \frac{1}{-1/n} = \sum_{n=1}^{\infty} (-1)^n (-n) = \sum_{n=1}^{\infty} (-1)^{n+1} n$。
	    这个级数的通项为 $a_n = (-1)^{n+1} n$。显然 $\lim_{n \to \infty} a_n \neq 0$。根据级数收敛的必要条件，该级数发散。

[[格林公式]]
- 这道题考察的是 #第二类曲线积分与路径无关的条件 ，这个知识点与 #格林公式 和 #保守场 （或称 #有势场 ） 密切相关。
[[第二类曲线积分与路径无关的条件]] 
在单连通区域 $D$ 内，以下四个命题是等价的：
1.  对于 $D$ 内的任意分段光滑闭合曲线 $C$，环路积分 $\oint_C Pdx + Qdy = 0$。
2.  对于 $D$ 内的任意两点 $A$ 和 $B$，曲线积分 $\int_L Pdx + Qdy$ 的值与路径 $L$ 无关，只取决于起点 $A$ 和终点 $B$。
3.  存在一个函数 $u(x,y)$（称为 #势函数 ），使得 $du = Pdx + Qdy$，即 $P = \frac{\partial u}{\partial x}$ 且 $Q = \frac{\partial u}{\partial y}$。
4.  函数 $P(x,y)$ 和 $Q(x,y)$ 在 $D$ 内具有一阶连续偏导数，并且满足条件 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$。
#分段光滑的简单闭合曲线 
1.   #分段光滑曲线的积分 
	1. 第一步：利用等价条件建立方程
		1. 满足条件 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$。
			我们先计算 $\frac{\partial Q}{\partial x}$。
			已知 $Q(x,y) = \frac{x}{y^2}$。将 $y$ 视为常数，对 $x$ 求偏导：
			$\frac{\partial Q}{\partial x} = \frac{\partial}{\partial x}(\frac{x}{y^2}) = \frac{1}{y^2}$
		2. 因此，我们需要的函数 $P(x,y)$ 必须满足：
			$\frac{\partial P}{\partial y} = \frac{1}{y^2}$
2. 第二步：检验四个选项 
	*   **(A)** $P(x,y) = y - \frac{x^2}{y^3} = y - x^2y^{-3}$
	    $\frac{\partial P}{\partial y} = \frac{\partial}{\partial y}(y - x^2y^{-3}) = 1 - x^2(-3y^{-4}) = 1 + \frac{3x^2}{y^4}$
	    这不等于 $\frac{1}{y^2}$，所以排除 A。
	*   **(B)** $P(x,y) = \frac{1}{y} - \frac{x^2}{y^3} = y^{-1} - x^2y^{-3}$
	    $\frac{\partial P}{\partial y} = \frac{\partial}{\partial y}(y^{-1} - x^2y^{-3}) = -y^{-2} - x^2(-3y^{-4}) = -\frac{1}{y^2} + \frac{3x^2}{y^4}$
	    这不等于 $\frac{1}{y^2}$，所以排除 B。
	*   **(C)** $P(x,y) = \frac{1}{x} - \frac{1}{y} = \frac{1}{x} - y^{-1}$
	    $\frac{\partial P}{\partial y} = \frac{\partial}{\partial y}(\frac{1}{x} - y^{-1}) = 0 - (-1y^{-2}) = \frac{1}{y^2}$
	    这个选项满足条件 $\frac{\partial P}{\partial y} = \frac{1}{y^2}$。
	*   **(D)** $P(x,y) = x - \frac{1}{y} = x - y^{-1}$
	    $\frac{\partial P}{\partial y} = \frac{\partial}{\partial y}(x - y^{-1}) = 0 - (-1y^{-2}) = \frac{1}{y^2}$
	    这个选项也满足条件 $\frac{\partial P}{\partial y} = \frac{1}{y^2}$。
3. 第三步：考虑函数的 #连续性 
	1. 等价条件成立的 #前提  函数 $P$ 和 $Q$ 及其一阶偏导数必须在整个区域内连续 
		*   区域是整个上半平面（$y > 0$）。
		*   对于选项 (C) $P(x,y) = \frac{1}{x} - \frac{1}{y}$，这个函数在 $x=0$（即 y 轴）上没有定义，是不连续的。而 y 轴是包含在“上半平面”这个区域内的。题目的条件是“对上半平面内的**任意**有向光滑闭曲线”，这个“任意”意味着曲线可以跨过 y 轴。因此，P(x,y) 必须在整个上半平面都连续，选项 (C) 不满足这个要求。

-   #找规律   
1. 证明单调性 
	1. $a_{n+1} - a_n = \int_{0}^{1} x^{n+1} \sqrt{1-x^2} dx - \int_{0}^{1} x^n \sqrt{1-x^2} dx$ 
	2. 利用定积分的线性性质，合并被积函数：
		$a_{n+1} - a_n = \int_{0}^{1} (x^{n+1} - x^n) \sqrt{1-x^2} dx = \int_{0}^{1} x^n(x-1) \sqrt{1-x^2} dx$
		1. 三者相乘得到$x^n(x-1) \sqrt{1-x^2} \le 0$
		2. 所以，$a_{n+1} - a_n < 0$，即$a_{n+1} < a_n$  #定积分的保号性   
		3. 数列{$a_n$}是单调递减的。 
2. 建立递推关系  
	1. #分部积分法   
	2. $a_n = \int_{0}^{1} x^{n-1} (x \sqrt{1-x^2}) dx$ 
		1. $v = \int x \sqrt{1-x^2} dx = -\frac{1}{2} \int \sqrt{1-x^2} d(1-x^2) = -\frac{1}{2} \frac{(1-x^2)^{3/2}}{3/2} = -\frac{1}{3}(1-x^2)^{3/2}$
	3. $a_n = \left[ u \cdot v \right]_0^1 - \int_0^1 v du$
		$a_n = \left[ x^{n-1} \cdot \left(-\frac{1}{3}(1-x^2)^{3/2}\right) \right]_0^1 - \int_0^1 \left(-\frac{1}{3}(1-x^2)^{3/2}\right) (n-1)x^{n-2} dx$
	4. 我们发现，这两个积分正是$a_{n-2}$和$a_n$的定义！
		$a_n = \frac{n-1}{3} (a_{n-2} - a_n)$
	5. $a_n = \frac{n-1}{n+2} a_{n-2}$ (其中$n=2, 3, \dots$) 
 -   (II) 求极限
1. [[夹逼准则（夹逼定理）]] 
2. 从$a_n < a_{n-1}$，两边同除以$a_{n-1}$ (因为$a_{n-1} > 0$)，得到：
    $\frac{a_n}{a_{n-1}} < 1$  
	1.  从$a_{n-1} < a_{n-2}$，两边取倒数，不等号反向：
	    $\frac{1}{a_{n-1}} > \frac{1}{a_{n-2}}$
	    将递推关系$a_n = \frac{n-1}{n+2} a_{n-2}$代入，得到$a_{n-2} = \frac{n+2}{n-1} a_n$。
	    代入不等式$a_{n-1} < a_{n-2}$中：
	    $a_{n-1} < \frac{n+2}{n-1} a_n$
	    两边同除以$a_{n-1}$并乘以$\frac{n-1}{n+2}$：
	    $\frac{n-1}{n+2} < \frac{a_n}{a_{n-1}}$
	2. 综合以上两个不等式，我们得到了一个夹逼结构：
		$\frac{n-1}{n+2} < \frac{a_n}{a_{n-1}} < 1$

[[形心的定义]] #形心坐标公式   
- 所以，求解这道题分为四个核心步骤：
	1.  确定$\bar{x}$的值（通常先考虑对称性）。
	2.  计算体积$V$。
	3.  计算关于$y$的一阶矩$\iiint_\Omega y \,dxdydz$。
	4.  计算关于$z$的一阶矩$\iiint_\Omega z \,dxdydz$。

1. 步骤一：利用对称性求$\bar{x}$    #利用对称性计算三重积分  
	1. 该立体由曲面$x^2 + (y-z)^2 = (1-z)^2$与平面$z=0$围成，且限制$0 \le z \le 1$
	2. **对称性分析**：我们观察立体的定义方程$x^2 + (y-z)^2 \le (1-z)^2$（因为是围成的“体”，所以是小于等于）。如果我们将$x$替换为$-x$，方程变为$(-x)^2 + (y-z)^2 = x^2 + (y-z)^2 \le (1-z)^2$，方程形式完全不变
	3. **结论**：这意味着立体$\Omega$关于$yOz$平面（即$x=0$平面）是对称的。对于对称的均匀物体，其形心必定位于对称面上。因此，形心的$x$坐标为0，即$\bar{x}=0$
2. 步骤二：计算体积$V$   $V = \iiint_\Omega dxdydz$ 
	1. **积分次序**：我们将积分次序定为先对$x,y$积分，再对$z$积分。即： #富比尼定理（交换积分次序）  
	    $V = \int_0^1 \left( \iint_{D_z} dxdy \right) dz$ 
	2. 计算体积$V = \iiint_\Omega dxdydz$。我们采用 #切片法 （或称截面法）来计算这个三重积分 [[三重积分的计算方法（切片法）（投影法）]]  
	3.  **计算截面面积**：$D_z$的面积$A(z)$为： 
	    $A(z) = \iint_{D_z} dxdy = \pi R^2 = \pi (1-z)^2$ 
	4. **对$z$积分**：将截面面积代入体积公式进行积分：
	    $V = \int_0^1 A(z) dz = \int_0^1 \pi (1-z)^2 dz$
	    $= \pi \left[ -\frac{(1-z)^3}{3} \right]_0^1$
	    $= \pi \left( -\frac{(1-1)^3}{3} - \left(-\frac{(1-0)^3}{3}\right) \right)$

		1. 对分段的分布函数求导：
			*   当 $z < 0$ 时，$f_Z(z) = \frac{d}{dz}(pe^z) = pe^z$。
			*   当 $z \ge 0$ 时，$f_Z(z) = \frac{d}{dz}(1 - (1-p)e^{-z}) = -(1-p)(-e^{-z}) = (1-p)e^{-z}$。
		2. 所以，Z 的概率密度函数为：
			$f_Z(z) = \begin{cases} pe^z & z < 0 \\ (1-p)e^{-z} & z \ge 0 \end{cases}$ 
	 **注**: 这个分布是 #拉普拉斯分布 (Laplace distribution) ，也叫 #双指数分布
-  (II) p 为何值时, X 与 Z 不相关  
1. 两个随机变量不相关的数学定义是它们的**协方差为零**，即 $Cov(X, Z) = 0$
	[[协方差]]  协方差的计算公式为：$Cov(X, Z) = E(XZ) - E(X)E(Z)$。  
	1.  **$E(X)$**: 已知 $X \sim E(1)$，所以 $E(X) = 1$ 
	2.  **$E(Z)$**: $E(Z) = E(XY)$。因为 X 和 Y 相互独立，所以 $E(XY) = E(X)E(Y)$。
	    *   $E(Y) = 1-2p$ (前面已计算)。
	    *   所以 $E(Z) = 1 \cdot (1-2p) = 1-2p$
	3. **$E(XZ)$**: 将 $Z=XY$ 代入，得到 $E(XZ) = E(X \cdot XY) = E(X^2Y)$。
	    *   因为 X 和 Y 相互独立，所以 $X^2$ 和 Y 也相互独立。
	    *   因此，$E(X^2Y) = E(X^2)E(Y)$。
	    *   $E(X^2) = 2$ (前面已计算)。
	    *   所以 $E(XZ) = 2 \cdot (1-2p) = 2 - 4p$。
		将这些值代入协方差公式：
		$Cov(X, Z) = E(XZ) - E(X)E(Z) = (2-4p) - 1 \cdot (1-2p) = 2 - 4p - 1 + 2p = 1 - 2p$
	令协方差为 0：
	$Cov(X, Z) = 1 - 2p = 0$
	解得 $p = \frac{1}{2}$
	4. 所以，当 $p = \frac{1}{2}$ 时，X 与 Z 不相关 
-  (III) X 与 Z 是否相互独立?  #判断是否相互独立  
	1. 检验是否对所有的 $(x, y)$ 都满足 $P(X=x, Y=y) = P(X=x)P(Y=y)$
	2. 我们来构造一个反例，就像解答中那样：
考虑事件 $\{X \le 1\}$ 和 $\{Z \le -1\}$。

1.  **计算联合概率 $P(X \le 1, Z \le -1)$**
    $P(X \le 1, Z \le -1) = P(X \le 1, XY \le -1)$
    因为 $X$ 是指数分布，所以 $X > 0$（严格来说是 $P(X>0)=1$）。
    为了使 $XY \le -1$ 成立，Y 必须为负，即 $Y=-1$。
    所以，这个事件可以改写为：
    $P(X \le 1, X(-1) \le -1, Y=-1) = P(X \le 1, -X \le -1, Y=-1)$
    $-X \le -1$ 等价于 $X \ge 1$。
    所以，事件变为 $P(X \le 1, X \ge 1, Y=-1)$。
    $X \le 1$ 和 $X \ge 1$ 同时成立的唯一可能是 $X=1$。
    因为 X 是连续型随机变量，它在任意单点的概率为 0，即 $P(X=1)=0$。
    因此，$P(X \le 1, Z \le -1) = P(X=1, Y=-1) = 0$。

2.  **计算概率的乘积 $P(X \le 1) \cdot P(Z \le -1)$**
    *   $P(X \le 1) = F_X(1) = 1 - e^{-1}$。这是一个大于 0 的数。
    *   $P(Z \le -1) = F_Z(-1)$。根据第一问的结果，当 $z < 0$ 时，$F_Z(z)=pe^z$。
    *   所以 $P(Z \le -1) = F_Z(-1) = pe^{-1}$。
    *   题目条件 $0 < p < 1$，所以 $pe^{-1}$ 也是一个大于 0 的数。

    因此，它们的乘积 $P(X \le 1)P(Z \le -1) = (1 - e^{-1})(pe^{-1}) > 0$。

3.  **比较结果**
    我们发现 $P(X \le 1, Z \le -1) = 0$，而 $P(X \le 1)P(Z \le -1) > 0$。
    两者不相等。
    $P(X \le 1, Z \le -1) \ne P(X \le 1)P(Z \le -1)$。

因此，X 与 Z **不相互独立**。这个结论对所有 $p \in (0,1)$ 都成立，包括使它们不相关的 $p = \frac{1}{2}$。

- 衍生 
	- 不相关与不独立的辨析
		- **关系**: 相互独立 $\implies$ 不相关。反之不成立。
	    *   **特例**: 如果 (X, Y) 服从**二维正态分布**，那么不相关与相互独立是等价的。这道题的 X 和 Z 显然不构成二维正态分布，所以不相关不意味着独立。
	    *   **考试形式**: 可能会直接问“不相关和独立的关系是什么？请举例说明。” 本题就是一个绝佳的例子
	* 随机变量函数的分布
		* #乘积分布: 本题是 $Z=XY$。
	    *   #和的分布 ($Z=X+Y$): 如果 X 和 Y 独立，其密度函数可以通过**卷积公式**求得：$f_Z(z) = \int_{-\infty}^{\infty} f_X(x)f_Y(z-x)dx$。
	    *   #商的分布 ($Z=Y/X$) 和 #最大/最小值分布 ($Z=\max(X,Y)$, $Z=\min(X,Y)$)  也是常见的考点。求解方法都与本题类似，从分布函数 $F_Z(z)$入手。
	* 题中用到的 $Cov(X, XY) = E(Y)D(X)$ 是一个可以记忆的技巧性结论（在 Y 与 X 独立时成立）

#概率密度函数的归一性  $\int_{-\infty}^{+\infty} f(x) dx = 1$ 
1. 题目给出的概率密度函数 $f(x; \sigma^2)$，我们知道当 $x < \mu$ 时，$f(x) = 0$，所以积分的下限从 $-\infty$ 变为 $\mu$。
    因此，我们有：
    $\int_{-\infty}^{+\infty} f(x; \sigma^2) dx = \int_{\mu}^{+\infty} \frac{A}{\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}} dx = 1$
2. #换元积分法   
	1. 令 $t = \frac{x-\mu}{\sigma}$。
	    对它求微分，得到 $dt = \frac{1}{\sigma} dx$。
	    同时，更新积分的上下限：
	    *   当 $x = \mu$ 时，$t = \frac{\mu-\mu}{\sigma} = 0$。
	    *   当 $x \to +\infty$ 时，$t \to +\infty$。
3. 转换积分 
	1. 结果代入原积分式，得到：
	    $1 = \int_{0}^{+\infty} A e^{-\frac{t^2}{2}} dt = A \int_{0}^{+\infty} e^{-\frac{t^2}{2}} dt$ 
4. 利用正态分布的性质求解 
	1. 知道标准正态分布 $N(0, 1)$ 的概率密度函数是 $\phi(z) = \frac{1}{\sqrt{2\pi}}e^{-\frac{z^2}{2}}$  
	2. 在 $(-\infty, +\infty)$ 上的积分为1，即 $\int_{-\infty}^{+\infty} \frac{1}{\sqrt{2\pi}}e^{-\frac{z^2}{2}} dz = 1$。
	    两边乘以 $\sqrt{2\pi}$，得到 $\int_{-\infty}^{+\infty} e^{-\frac{z^2}{2}} dz = \sqrt{2\pi}$。
	    由于 $e^{-\frac{z^2}{2}}$ 是一个偶函数（关于y轴对称），所以它在 $(0, +\infty)$ 上的积分等于在 $(-\infty, +\infty)$ 上积分的一半。
	    因此，$\int_{0}^{+\infty} e^{-\frac{t^2}{2}} dt = \frac{1}{2} \int_{-\infty}^{+\infty} e^{-\frac{t^2}{2}} dt = \frac{\sqrt{2\pi}}{2}$
5. **解出 A**：
    将上一步的结果代入第3步的式子中：
    $1 = A \cdot \frac{\sqrt{2\pi}}{2}$
    解得 $A = \frac{2}{\sqrt{2\pi}} = \sqrt{\frac{4}{2\pi}} = \sqrt{\frac{2}{\pi}}$。
    这与图片中的答案完全一致。
-  (II): 求 $\sigma^2$ 的最大似然估计量 
1.  [[最大似然估计法]] 
	1. 第一步：写出似然函数 $L(\sigma^2)$
		1. $L(\sigma^2) = \prod_{i=1}^{n} f(x_i; \sigma^2) = \prod_{i=1}^{n} \left( \frac{A}{\sigma} e^{-\frac{(x_i-\mu)^2}{2\sigma^2}} \right)$
		    将第一问求出的 $A = \sqrt{\frac{2}{\pi}}$ 和 $\sigma = \sqrt{\sigma^2}$ 代入：
		    $L(\sigma^2) = \prod_{i=1}^{n} \left( \frac{\sqrt{2/\pi}}{\sqrt{\sigma^2}} e^{-\frac{(x_i-\mu)^2}{2\sigma^2}} \right)$
		    $L(\sigma^2) = \left( \sqrt{\frac{2}{\pi}} \right)^n \left( \frac{1}{\sqrt{\sigma^2}} \right)^n \prod_{i=1}^{n} e^{-\frac{(x_i-\mu)^2}{2\sigma^2}}$
		    $L(\sigma^2) = \left( \frac{2}{\pi} \right)^{n/2} (\sigma^2)^{-n/2} e^{-\frac{1}{2\sigma^2}\sum_{i=1}^{n}(x_i-\mu)^2}$
	2. 第二步：取对数，得到对数似然函数 $\ln L(\sigma^2)$
		1. 取对数可以把连乘和指数运算转化为加和与乘法，极大简化求导。
		    $\ln L(\sigma^2) = \ln \left[ \left( \frac{2}{\pi} \right)^{n/2} (\sigma^2)^{-n/2} e^{-\frac{1}{2\sigma^2}\sum_{i=1}^{n}(x_i-\mu)^2} \right]$
		    $\ln L(\sigma^2) = \ln\left(\frac{2}{\pi}\right)^{n/2} + \ln(\sigma^2)^{-n/2} + \ln e^{-\frac{1}{2\sigma^2}\sum_{i=1}^{n}(x_i-\mu)^2}$
		    $\ln L(\sigma^2) = \frac{n}{2}\ln\left(\frac{2}{\pi}\right) - \frac{n}{2}\ln(\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^{n}(x_i-\mu)^2$
		    这与图片中的对数似然函数一致。
	3. 第三步：求导并令其为零 
		1. 将对数似然函数对未知参数 $\sigma^2$ 求导。注意，这里是把 $\sigma^2$ 看作一个整体变量。
		    $\frac{d[\ln L(\sigma^2)]}{d\sigma^2} = \frac{d}{d\sigma^2} \left[ \frac{n}{2}\ln\left(\frac{2}{\pi}\right) - \frac{n}{2}\ln(\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^{n}(x_i-\mu)^2 \right]$
		    第一项是常数，导数为0。第二项 $\frac{d}{d\theta}(-\frac{n}{2}\ln\theta) = -\frac{n}{2\theta}$。第三项 $\frac{d}{d\theta}(-C \cdot \theta^{-1}) = C \cdot \theta^{-2}$。
		    $\frac{d[\ln L(\sigma^2)]}{d\sigma^2} = 0 - \frac{n}{2} \cdot \frac{1}{\sigma^2} - \left(-\frac{1}{2(\sigma^2)^2}\right) \sum_{i=1}^{n}(x_i-\mu)^2$
		    $\frac{d[\ln L(\sigma^2)]}{d\sigma^2} = -\frac{n}{2\sigma^2} + \frac{1}{2\sigma^4}\sum_{i=1}^{n}(x_i-\mu)^2$
		    令导数为0：
		    $-\frac{n}{2\sigma^2} + \frac{1}{2\sigma^4}\sum_{i=1}^{n}(x_i-\mu)^2 = 0$
		    两边同乘以 $2\sigma^4$：
		    $-n\sigma^2 + \sum_{i=1}^{n}(x_i-\mu)^2 = 0$
		    整理得到 $\sigma^2$ 的估计值 $\hat{\sigma}^2$：
		    $\hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^{n}(x_i-\mu)^2$
	4. **得出最大似然估计量**
		将样本观测值 $x_i$ 替换为随机变量 $X_i$，就得到了 $\sigma^2$ 的最大似然估计量 (MLE)：
	    $\hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^{n}(X_i-\mu)^2$ 
- 衍生 
	- [[矩估计法]] 
		- 这是另一种常用的点估计方法。考题可能会要求你同时用最大似然法和矩估计法求解，并比较二者结果。矩估计法的思想是令样本矩等于总体矩。例如，令样本一阶矩（样本均值）等于总体一阶矩（期望）。
	- $\mu$ 未知的情况 
		- 本题中 $\mu$ 是已知参数。一个非常经典的变式是 $\mu$ 和 $\sigma^2$ 均未知。此时需要求解一个参数方程组，即分别对 $\mu$ 和 $\sigma^2$ 求偏导，并令它们都等于0。通常情况下，$\mu$ 的最大似然估计量是样本均值 $\bar{X}$，而 $\sigma^2$ 的最大似然估计量是 $\frac{1}{n}\sum_{i=1}^{n}(X_i-\bar{X})^2$。 
	-  分布的截断问题 
		- 本题给出的其实是一个在 $x=\mu$ 处截断的“半个”正态分布。这类截断分布 (Truncated Distribution) 的期望和方差计算会与标准正态分布有所不同，这也是一个潜在的考点。
