掌握常用幂级数展开式及其**收敛域**是考研数学或微积分考试中的基本功。

为了方便记忆，我把它们分成了三类：**“全能型”**（全轴收敛）、**“基础型”**（开区间）和**“边界敏感型”**（需注意端点）。

---

### 第一类：全能型（收敛域为无穷大）
这三个函数的级数在实数轴上任何地方都收敛，不需要考虑端点问题。

| 函数 | 展开式 (麦克劳林) | 收敛域 | 备注 |
| :--- | :--- | :--- | :--- |
| **$e^x$** | $\sum_{n=0}^{\infty} \frac{x^n}{n!}$ | $(-\infty, +\infty)$ | 随便代，都收敛 |
| **$\sin x$** | $\sum_{n=0}^{\infty} (-1)^n \frac{x^{2n+1}}{(2n+1)!}$ | $(-\infty, +\infty)$ | 只有奇次项 |
| **$\cos x$** | $\sum_{n=0}^{\infty} (-1)^n \frac{x^{2n}}{(2n)!}$ | $(-\infty, +\infty)$ | 只有偶次项 |

---

### 第二类：基础型（收敛域为开区间）
这类级数来源于等比级数（几何级数），两端点通常都发散，或者无定义。

| 函数 | 展开式 | 收敛域 | 记忆点 |
| :--- | :--- | :--- | :--- |
| **$\frac{1}{1-x}$** | $\sum_{n=0}^{\infty} x^n$ | $(-1, 1)$ | $x=1$ 分母为0；$x=-1$ 级数震荡 |
| **$\frac{1}{1+x}$** | $\sum_{n=0}^{\infty} (-1)^n x^n$ | $(-1, 1)$ | 同上 |

---

### 第三类：边界敏感型（收敛半径是1，但端点情况不同）
这是考试最喜欢挖坑的地方，重点是**端点是否包含**。

| 函数 | 展开式 | 收敛域 | 端点分析 (为什么?) |
| :--- | :--- | :--- | :--- |
| **$\ln(1+x)$** | $\sum_{n=1}^{\infty} (-1)^{n-1} \frac{x^n}{n}$ | **$(-1, 1]$** | **左开右闭**。<br>右端 $x=1$ 是交错级数(收敛)；<br>左端 $x=-1$ 是调和级数(发散)。 |
| **$\arctan x$** | $\sum_{n=0}^{\infty} (-1)^n \frac{x^{2n+1}}{2n+1}$ | **$[-1, 1]$** | **两端均闭**。<br>$x=\pm 1$ 时均为交错级数，均收敛。 |
| **$\arcsin x$** | $\sum_{n=0}^{\infty} \frac{(2n)!}{4^n (n!)^2 (2n+1)} x^{2n+1}$ | **$[-1, 1]$** | **两端均闭**。<br>虽然看起来复杂，但在端点处收敛（绝对收敛）。 |
| **$(1+x)^\alpha$** | $1 + \alpha x + \frac{\alpha(\alpha-1)}{2!}x^2 + \dots$ | **情况复杂** | 基本区间是 $(-1, 1)$，端点取决于 $\alpha$：<br>1. $\alpha > 0$: $[-1, 1]$ (如 $\sqrt{1+x}$)<br>2. $-1 < \alpha < 0$: $(-1, 1]$ (如 $\frac{1}{\sqrt{1+x}}$)<br>3. $\alpha \le -1$: $(-1, 1)$ (如 $\frac{1}{1+x}$) |

---

### 🚀 必杀技：如何用这些公式秒杀题目？

题目通常不会直接考 $x$，而是考 $2x$, $x/2$, $x^2$ 等等。只要把那一坨东西看作一个**整体 $\mathbf{u}$**。

**通用步骤：**

1.  **确定母版公式：** 看函数长得像谁。
    *   比如 $f(x) = \frac{1}{1 - 3x}$，长得像 $\frac{1}{1-\mathbf{u}}$。
2.  **提取整体 $\mathbf{u}$：** 这里 $\mathbf{u} = 3x$。
3.  **套用母版收敛域：**
    *   母版 $\frac{1}{1-\mathbf{u}}$ 的域是 $-1 < \mathbf{u} < 1$。
4.  **解不等式：**
    *   $-1 < 3x < 1 \Rightarrow -\frac{1}{3} < x < \frac{1}{3}$。
    *   收敛区间即为 $(-\frac{1}{3}, \frac{1}{3})$。

**再举个例子（端点情况）：**
若 $f(x) = \ln(1 - x)$。
1.  **母版：** $\ln(1+\mathbf{u})$，收敛域 $-1 < \mathbf{u} \le 1$。
2.  **代换：** 这里 $\mathbf{u} = -x$。
3.  **解不等式：** $-1 < -x \le 1$。
4.  **变号变向：** $1 > x \ge -1$，即 $[-1, 1)$。
    *(看，区间从“左开右闭”变成了“左闭右开”，就是因为那个负号)*

**总结记忆口诀：**
> 指数三角无穷大，
> 等比级数开区间。
> 对数左开右边闭，
> 反切两头都能取。
