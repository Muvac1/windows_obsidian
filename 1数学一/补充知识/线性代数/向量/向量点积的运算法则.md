*   **在二维空间中：**
    如果向量 **a** = (x₁, y₁) ，向量 **b** = (x₂, y₂)，则：
    **a** · **b** = x₁x₂ + y₁y₂

*   **在三维空间中：**
    如果向量 **a** = (x₁, y₁, z₁) ，向量 **b** = (x₂, y₂, z₂)，则：
    **a** · **b** = x₁x₂ + y₁y₂ + z₁z₂

*   **在 n 维空间中：**
    如果向量 **a** = (a₁, a₂, ..., aₙ) ，向量 **b** = (b₁, b₂, ..., bₙ)，则：
    **a** · **b** = a₁b₁ + a₂b₂ + ... + aₙbₙ 

#### 2. 几何定义

这种定义揭示了点积的物理和几何意义。

**a** · **b** = |**a**| |**b**| cos(θ)

其中：
*   |**a**| 是向量 **a** 的模（长度）。
*   |**b**| 是向量 **b** 的模（长度）。
*   θ 是两个向量起点重合时，它们之间的夹角（取值范围为 0 ≤ θ ≤ π 或 0° ≤ θ ≤ 180°）。
