### 考点一：单矩阵的“正交对角化”
**场景**：给一个 #实对称矩阵 $A$（或一个二次型 $f$），求正交矩阵 $Q$，使得 $Q^T A Q = \Lambda$（对角阵）。
这是最基础的题型，是所有后续题型的地基。 [[实对称矩阵的性质]]

*   **解法**：**标准流程法**
    1.  求特征值 $|\lambda I - A| = 0$。
    2.  解方程 $(A-\lambda I)x=0$ 求特征向量。
    3.  **正交化**：
        *   若特征值不同：天然正交，直接下一步。
        *   若有重根：**必须**用施密特正交化（Gram-Schmidt）。
        *   *技巧*：如果是 3阶矩阵且求最后列，可用**叉乘法**。
    4.  **单位化**：所有向量除以模长。
    5.  组装 $Q$。

### 考点二：双矩阵的“正交相似变换” (即本题考法)
**场景**：给两个实对称矩阵 $A$ 和 $B$，已知它们相似（或让你证明相似），求正交矩阵 $Q$ 使得 $B = Q^T A Q$。

*   **解法**：**桥梁传递法** (你刚刚用的那个)  
    *   **核心公式**：$Q = Q_A Q_B^T$
    *   **步骤**：
        1.  对 $A$ 进行对角化，求出 $Q_A$，使得 $Q_A^T A Q_A = \Lambda$。
        2.  对 $B$ 进行对角化，求出 $Q_B$，使得 $Q_B^T B Q_B = \Lambda$。
    *   **致命坑点（必考）**：**特征值的顺序必须一致！**
        *   如果 $\Lambda = \text{diag}(1, 1, 5)$，那么 $Q_A$ 和 $Q_B$ 的前两列必须对应特征值 1，第三列必须对应特征值 5。顺序一乱，算出来的 $Q$ 就是错的。
        * [[双矩阵的“正交相似变换”例题]]

### 考点三：已知部分向量，构造正交矩阵 (填空/小题)
**场景**：题目只给了你一个向量 $\alpha = (1, 1, 1)^T$，让你构造一个 3阶正交矩阵 $Q$，把 $\alpha$ 作为第一列（或 $Q$ 的第一列是 $\alpha$ 的单位化）。
 [[已知部分向量，构造正交矩阵 (填空小题)例题]] 
*   **解法 1：施密特补全法 (通用)**
    1.  写出 $\alpha$ (记为 $\beta_1$)。
    2.  随意找两个与 $\beta_1$ 线性无关的向量（比如 $\beta_2=(1,0,0)^T$, $\beta_3=(0,1,0)^T$，只要不平行就行）。
    3.  对 $\beta_1, \beta_2, \beta_3$ 进行施密特正交化。
    4.  单位化。

*   **解法 2：齐次方程法 (推荐，更快)**
    1.  要找与 $\alpha=(1,1,1)^T$ 正交的向量，相当于解方程 $1x_1 + 1x_2 + 1x_3 = 0$。
    2.  这个方程的基础解系有两个，比如 $\xi_1 = (-1, 1, 0)^T$ 和 $\xi_2 = (-1, 0, 1)^T$。
    3.  现在你有三个向量了：$\alpha, \xi_1, \xi_2$。
    4.  注意：$\xi_1$ 和 $\xi_2$ 不一定正交，需要对这两个做一次施密特正交化。
    5.  单位化，搞定。

### 考点四：反求矩阵 A (逆向思维)
**场景**：已知 $A$ 的特征值 $\lambda_1, \lambda_2, \lambda_3$ 和对应的特征向量 $\alpha_1, \alpha_2, \alpha_3$（已正交单位化），求矩阵 $A$。

*   **解法**： #谱分解公式 
    不需要先把 $Q$ 拼出来再算 $Q \Lambda Q^T$（矩阵乘法太慢）。
    直接用公式：
    $$A = \lambda_1 \alpha_1 \alpha_1^T + \lambda_2 \alpha_2 \alpha_2^T + \lambda_3 \alpha_3 \alpha_3^T$$
    *   **原理**：这是 $A = Q \Lambda Q^T$ 的展开形式。
    *   **计算**：列向量乘以行向量得到一个矩阵，然后三个矩阵相加。速度极快。
[[反求矩阵 A (逆向思维)例题]]

### 考点五：既非相似也非合同，单纯求旋转 (几何考法)
**场景**：在解析几何或二次型化简中，问你“经过怎样的坐标变换，消去交叉项”。

*   **解法**：
    这本质上就是**考点一**。
    只是问法变了。要求的坐标变换 $\mathbf{x} = Q\mathbf{y}$ 中的 $Q$，就是**考点一**里求出的那个 $Q$。
    *   **注意**：有时候题目要求 $|Q|=1$（代表旋转，不含反射）。
    *   如果你求出的 $Q$ 行列式是 -1（比如你特征向量方向选反了），只需要把 $Q$ 的任意一列乘以 -1，行列式就变成 1 了。

### 总结：考试遇到“求正交矩阵”怎么反应？

1.  **看题目给几个矩阵？**
    *   **给一个 A** $\rightarrow$ **考点一**（直接求特征值特征向量）。
    *   **给 A 和 B** $\rightarrow$ **考点二**（桥梁法，$Q=Q_A Q_B^T$，注意顺序）。
    *   **给特征值/向量求 A** $\rightarrow$ **考点四**（谱分解公式）。
    *   **给一个向量求 Q** $\rightarrow$ **考点三**（解方程找垂线 + 施密特）。

你现在的进度已经掌握了最难的考点二（双矩阵），只要注意**“特征值顺序对齐”**这个细节，这块内容你就已经通关了！
