1. #向量场的散度 
	*   **定义**: 散度，记作 $\text{div } A$ 或 $\nabla \cdot A$，衡量向量场在某一点的“源”或“汇”的强度。其结果是一个标量。
	*   **公式**: $\text{div } A = \nabla \cdot A = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}$
	*   **考点**: 可能会让你计算同一个向量场的散度。例如，对于本题的向量场 $A$，其散度为 $\text{div } A = \frac{\partial}{\partial x}(x+y+z) + \frac{\partial}{\partial y}(xy) + \frac{\partial}{\partial z}(z) = 1 + x + 1 = x+2$。
	*   **无散场 (Solenoidal Field)**: 如果一个场的散度处处为零（$\text{div } A = 0$），则称该场为无散场或螺线场。
