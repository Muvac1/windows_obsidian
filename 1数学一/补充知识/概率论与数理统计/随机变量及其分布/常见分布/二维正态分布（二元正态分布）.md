#二维正态分布  
对于服从二维正态分布的两个随机变量，它们相互独立的充要条件是它们的相关系数为0（即$\rho=0$）

*   一元正态分布是在一条**线**上的“钟形曲线”。
*   二元正态分布是在一个**平面**（X轴和Y轴）上的“立体钟形山”。

一个二元正态分布由 **5个参数** 唯一确定，记作 $(X, Y) \sim N(\mu_1, \mu_2; \sigma_1^2, \sigma_2^2; \rho)$。

1.  **$\mu_1$**：$X$ 的中心位置（山峰在X轴的坐标）。
2.  **$\mu_2$**：$Y$ 的中心位置（山峰在Y轴的坐标）。
3.  **$\sigma_1$**：$X$ 的波动大小（山包沿着X轴方向的胖瘦）。
4.  **$\sigma_2$**：$Y$ 的波动大小（山包沿着Y轴方向的胖瘦）。
5.  **$\rho$ (关键)**：$X$ 和 $Y$ 的**相关系数**。它决定了山包从“俯视”看下去的形状和方向。

$\rho$ 是二元正态分布的灵魂，它的取值范围是 $[-1, 1]$。想象你从高空俯视这座“钟形山”，看它的等高线（底座）：

*   **当 $\rho = 0$ 时**：$X$ 和 $Y$ 不相关。
    *   如果 $\sigma_1 = \sigma_2$，俯视图是**正圆**。
    *   如果 $\sigma_1 \neq \sigma_2$，俯视图是**正椭圆**（长轴平行于坐标轴）。
*   **当 $\rho > 0$ 时**：$X$ 和 $Y$ 正相关。
    *   俯视图是**倾斜的椭圆**，长轴像“/”一样向右上方倾斜。$\rho$ 越接近1，椭圆越扁长，X变大时Y也倾向于变大。
*   **当 $\rho < 0$ 时**：$X$ 和 $Y$ 负相关。
    *   俯视图是**倾斜的椭圆**，长轴像“\”一样向右下方倾斜。X变大时Y倾向于变小。

### 3. 三大核心性质（考试必考）

这是做题时最常用的工具箱，一定要记住：

#### 性质一：边缘分布也是正态分布
如果你只看 $X$（忽略 $Y$），或者只看 $Y$（忽略 $X$），它们各自都是正态分布。
*   $X \sim N(\mu_1, \sigma_1^2)$
*   $Y \sim N(\mu_2, \sigma_2^2)$
*   *考点*：题目给你二元正态分布，让你求 $X$ 单独的概率，直接用一元正态公式即可。

#### 性质二：独立性等价于不相关（特有性质！）
在一般的统计学中，“不相关（Cov=0）”不一定代表“独立”。**但在正态分布中，这是一回事！**
*   **结论**：对于二元正态分布，**$\rho = 0$ $\iff$ $X$ 与 $Y$ 相互独立**。
*   *考点*：题目问“什么条件下X和Y独立？”，答案就是 $\rho=0$ 或 $\text{Cov}(X,Y)=0$。

#### 性质三：线性组合仍为正态分布（刚才那道题的原理）
如果 $(X, Y)$ 服从二元正态分布，那么它们的任意线性组合 $Z = aX + bY$ 也服从一元正态分布。
我们需要确定 $Z$ 的期望和方差：
*   $E(Z) = a\mu_1 + b\mu_2$
*   $D(Z) = a^2\sigma_1^2 + b^2\sigma_2^2 + 2ab\rho\sigma_1\sigma_2$ （注意这里的交叉项！）

### 4. 刚才那道题为什么要用这个？

在刚才的题目中，我们求的是 $\hat{\theta} = \bar{X} - \bar{Y}$ 的方差。
这就是典型的 **线性组合** 问题，其中 $a=1, b=-1$。

根据性质三的方差公式：
$$D(X - Y) = 1^2 \cdot \sigma_1^2 + (-1)^2 \cdot \sigma_2^2 + 2(1)(-1)\text{Cov}(X,Y)$$
$$D(X - Y) = \sigma_1^2 + \sigma_2^2 - 2\text{Cov}(X,Y)$$

因为题目说样本来自二元正态分布，已知相关系数为 $\rho$，所以：
$$\text{Cov}(X,Y) = \rho \sigma_1 \sigma_2$$

这就是为什么最终公式里会出现 $-2\rho\sigma_1\sigma_2$ 这一项。如果是独立的（$\rho=0$），这一项就没有了。

### 5. 常见的考题陷阱/考法

1.  **条件分布**：
    *   题目问：已知 $Y = y$ 的情况下，$X$ 的分布是什么？
    *   *结论*：条件分布 $(X|Y=y)$ 依然是**正态分布**。
    *   *公式（了解即可，考研或高级统计会考）*：
        *   均值会偏移：$E(X|Y=y) = \mu_1 + \rho \frac{\sigma_1}{\sigma_2}(y - \mu_2)$ （回归直线）
        *   方差会变小：$D(X|Y=y) = \sigma_1^2 (1 - \rho^2)$

2.  **判断独立性**：
    *   题目给出一个复杂的联合概率密度公式 $f(x,y) = K \cdot \exp\{ -[\dots] \}$，问你 $X$ 和 $Y$ 是否独立？
    *   *技巧*：看指数部分是否有 $xy$ 的交叉项。如果没有交叉项（即形式可以拆分为 $g(x)h(y)$），则 $\rho=0$，独立；如果有交叉项，则不独立。

### 总结

对于二元正态分布，你只需要掌握：
1.  **5个参数**：两个均值，两个方差，一个相关系数。
2.  **几何形状**：俯视是椭圆，$\rho$ 决定扁平程度和方向。
3.  **计算方差**：遇到 $X \pm Y$ 的方差，千万别忘了 $\pm 2\rho\sigma_1\sigma_2$ 这一项。
4.  **黄金定律**：在正态分布的世界里，不相关（$\rho=0$）就是独立。
