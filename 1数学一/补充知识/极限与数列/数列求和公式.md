#等差数列求和公式 
1. **基本公式：**  
    **Sₙ = $\frac{n}{2}$ * (a₁ + aₙ)**  
    (和 = 项数/2 * (首项 + 末项)
2. **常用公式：**  
    **Sₙ = n*a₁ + n(n-1)/2 * d**  
	(将 aₙ = a₁ + (n-1)d 代入上式得到)
 
 #等比数列求和公式 
对于首项为 a₁，公比为 q，项数为 n 的等比数列：

1. 当 **q ≠ 1** 时：  
    **Sₙ =$\frac{a_{1}(1 - q^n)}{(1 - q)}$ 
2. 当 **q = 1** 时：  
    **Sₙ = n * a₁**
### 其他常见的数列求和公式

除了等差和等比数列，还有一些重要的求和公式：
1. **自然数数列和：**  
    1 + 2 + 3 + ... + n = **n(n+1)/2**
    
2. **自然数平方和：**  
    1² + 2² + 3² + ... + n² = **n(n+1)(2n+1)/6**
    
3. **自然数立方和：**  
    1³ + 2³ + 3³ + ... + n³ = **[n(n+1)/2]²**
