[[不同类型的幂级数求和]]
### 一、 幂级数和函数 $S(x)$ 的通用求法

核心思想是：**通过“逐项求导”或“逐项积分”，将未知的级数转化为已知的基本初等函数展开式（主要是几何级数 $\frac{1}{1-x}$）。**

#### 1. 基本步骤“三部曲”

1.  **求收敛域**：
    *   使用**比值判别法** $\lim_{n \to \infty} |\frac{a_{n+1}}{a_n}| = \rho$。
    *   收敛半径 $R = \frac{1}{\rho}$。
    *   **必须**单独讨论端点 $x = \pm R$ 处的敛散性，确定收敛域。
2.  **变形与转化**（核心步骤）：
    *   目标通常是凑出 $\sum x^n$ 的形式，因为我们知道 $\sum_{n=0}^\infty x^n = \frac{1}{1-x}$。
    *   **消去系数中的 $n$**：
        *   如果 $n$ 在**分子**上（如 $\sum n x^n$），通常通过**逐项求导**，或者提取 $x$ 后求导。
        *   如果 $n$ 在**分母**上（如 $\sum \frac{x^n}{n}$），通常通过**逐项积分**，或者除以 $x$ 后积分。
3.  **还原与定界**：
    *   如果用了求导，最后要积分回去；如果用了积分，最后要求导回去。
    *   **注意常数项**：积分时会出现常数 $C$，通常代入 $x=0$ 来确定 $C$。
    *   **注意定义域**：写出 $S(x)$ 的解析式后，要标明 $x$ 的取值范围（即收敛域）。

#### 2. 常见操作技巧

*   **提公因式**： $\sum_{n=1}^\infty n x^{n+1} = x^2 \sum_{n=1}^\infty n x^{n-1}$。
*   **下标变换**： 求导或积分后，求和符号的起始项 $n$ 可能会变化（例如从 $n=0$ 变为 $n=1$），需要小心处理。
*   **拆分**： $\sum (n^2+n) x^n = \sum n^2 x^n + \sum n x^n$，分别求解。
*   **阶乘处理**： 如果系数包含 $n!$，通常往 $e^x, \sin x, \cos x$ 的方向凑。

### 二、 必背的“六大”基本展开式

如果不背这些，求和函数就是无米之炊。

1.  $\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n, \quad (-1 < x < 1)$ **(万能公式，最常用)**
2.  $\frac{1}{1+x} = \sum_{n=0}^{\infty} (-1)^n x^n, \quad (-1 < x < 1)$
3.  $e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!}, \quad (-\infty < x < +\infty)$
4.  $\ln(1+x) = \sum_{n=1}^{\infty} (-1)^{n-1} \frac{x^n}{n}, \quad (-1 < x \le 1)$
5.  $\arctan x = \sum_{n=0}^{\infty} (-1)^n \frac{x^{2n+1}}{2n+1}, \quad [-1 \le x \le 1]$
6.  $(1+x)^\alpha = 1 + \alpha x + \frac{\alpha(\alpha-1)}{2!}x^2 + \dots \quad (\text{二项式展开})$

### 三、 相关的核心知识点与考法

除了直接求 $S(x)$，以下知识点常结合考查：

#### 1. 数项级数求和 (利用 $S(x)$ 求特殊值)
*   **考法**：题目不直接让你求函数，而是让你求一个常数级数的和，例如 $\sum_{n=1}^{\infty} \frac{n}{2^n}$。
*   **解法**：
    1.  构造幂级数 $\sum n x^n$。
    2.  求出和函数 $S(x) = \frac{x}{(1-x)^2}$。
    3.  令 $x = \frac{1}{2}$ 代入 $S(x)$ 算出结果。

#### 2. 阿贝尔定理 (Abel's Theorem) 与端点连续性
*   **难点**：当你求出的和函数 $S(x)$ 是在开区间 $(-R, R)$ 内得到的（比如利用了积分法），但题目让你求端点处的级数和（比如 $x=R$ 或 $x=-R$）。
*   **知识点**：若幂级数在端点 $x=R$ 处收敛，则和函数 $S(x)$ 在 $x=R$ 处通常是**左连续**的。
    *   即：$\sum_{n=0}^\infty a_n R^n = \lim_{x \to R^-} S(x)$。
    *   *应用场景*：求 $\sum_{n=1}^\infty \frac{(-1)^{n-1}}{n}$。你知道 $\sum \frac{(-1)^{n-1}x^n}{n} = \ln(1+x)$ 在 $(-1, 1)$ 成立。根据连续性，代入 $x=1$，得级数和为 $\ln 2$。

#### 3. 微分方程法
*   **考法**：有些级数的系数递推关系很复杂，或者很难直接凑。
*   **解法**：设 $y = S(x)$，通过对级数求导，建立 $y', y'', y$ 之间的关系，构造一个**微分方程**。
*   *例子*：若 $S(x) = \sum \frac{x^n}{n!}$，求导发现 $S'(x) = S(x)$，且 $S(0)=1$，解得 $S(x)=e^x$。

#### 4. 狄利克雷判别法 (Dirichlet Test)
*   虽然这是常数级数的判别法，但在讨论**幂级数收敛域端点**时非常有用。如果端点处级数是交错级数，常用莱布尼茨判别法；如果是三角级数形式，可能用到狄利克雷。

### 四、 实战举例 (典型题模型)

**题目**：求幂级数 $\sum_{n=1}^{\infty} n x^n$ 的和函数。

**思路流程**：
1.  **收敛半径**：$\lim |\frac{(n+1)}{n}| = 1$，故 $R=1$。区间 $(-1, 1)$。
2.  **观察**：系数有个 $n$，分子是一次幂，想到求导。
3.  **构造**：
    $$S(x) = \sum_{n=1}^{\infty} n x^n = x \sum_{n=1}^{\infty} n x^{n-1}$$
    注意到 $(x^n)' = n x^{n-1}$，所以：
    $$S(x) = x \left( \sum_{n=0}^{\infty} x^n \right)'$$
    *(注：$n=1$求导后变成常数项，求和可以从0开始或保持1，这里 $\sum x^n$ 求导即为所求)*
4.  **代入已知公式**：
    已知 $\sum_{n=0}^{\infty} x^n = \frac{1}{1-x}$。
5.  **计算**：
    $$S(x) = x \left( \frac{1}{1-x} \right)' = x \cdot \frac{1}{(1-x)^2} = \frac{x}{(1-x)^2}$$
6.  **结论**：
    $$S(x) = \frac{x}{(1-x)^2}, \quad x \in (-1, 1)$$

### 总结
遇到“求幂级数和函数”：
1.  先求收敛域（别忘了端点）。
2.  看系数：$n$ 在分子 $\to$ 求导；$n$ 在分母 $\to$ 积分。
3.  凑几何级数 $\frac{1}{1-x}$。
4.  最后反向运算还原，$x=0$ 定常数。
