信号量由Dijkstra提出的经典同步工具。可以使用一个初值为1的 #二元信号量（互斥锁，Mutex） 来保护临界区。
        ```
        semaphore mutex = 1;

        // P1
        P(mutex);       // 等待并锁住
        load R1, x;
        inc R1;
        store x, R1;
        V(mutex);       // 释放锁

        // P2
        P(mutex);       // 等待并锁住
        load R2, x;
        dec R2;
        store x, R2;
        V(mutex);       // 释放锁
        ```
        使用信号量后，两个临界区代码段将互斥执行，结果将始终为1。

*   **锁(Lock)**: 更现代的编程语言和库中提供的同步原语，功能与互斥信号量类似。

*   **硬件指令支持**: 如`TestAndSet`、`CompareAndSwap`等原子指令，它们是实现锁和信号量的底层基础。
