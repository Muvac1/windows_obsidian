### 核心知识点： #MIPS的三种指令格式  

这堆简写（`rd`, `rs`, `rt`, `imm`）背后的真正考点是 **MIPS指令格式（Instruction Format）**。MIPS指令长度固定为32位，为了凑齐这32位，设计者规定了三种“拼图方式”：

#### 1. R型指令 (Register Type) - 用于运算
*   **长相**：`op rd, rs, rt` （例如：`addu $1, $2, $3`）
*   **含义**：
    *   **`rs` (Source)**: 第一个源操作数寄存器。
    *   **`rt` (Target)**: 第二个源操作数寄存器（注意：在R型里它是来源！）。
    *   **`rd` (Destination)**: **目的寄存器**，存放运算结果。
*   **考点**：如果你把结果写到了 `rt` 而不是 `rd` 里，在画数据通路（Datapath）时，你的线就连错了。

#### 2. I型指令 (Immediate Type) - 用于访存和立即数运算
*   **长相**：`op rt, rs, imm` （例如：`lw $1, 100($2)` 或 `ori $1, $2, 10`）
*   **含义**：
    *   **`rs`**: 源操作数（通常是基址）。
    *   **`imm`**: 立即数（常数/偏移量）。
    *   **`rt`**: **既可能是源，也可能是目的**（这是最大的坑！）。
        *   对于 `lw` (Load Word)：`rt` 是**目的**（接收内存数据）。
        *   对于 `sw` (Store Word)：`rt` 是**源**（把这里的数据存进内存）。
        *   对于 `ori`/`addi`：`rt` 是**目的**（接收运算结果）。

#### 3. J型指令 (Jump Type) - 用于跳转
*   **长相**：`op target`
*   **含义**：直接跳去某个地址，不涉及寄存器读写。

### 考试是怎么考的？（常见题型）

这个知识点通常隐藏在以下几种题目里：

#### 考法一：数据通路与控制信号（就像你发的这道题）
*   **题目样子**：给一个指令 `lw rt, offset(rs)`，问你 `RegDst`（目标寄存器选择）信号是 0 还是 1？
*   **解题逻辑**：
    *   你必须知道：`lw` 是 I型指令。
    *   I型指令的目标寄存器位置在指令的 16-20位（即 `rt` 位置）。
    *   R型指令的目标寄存器位置在指令的 11-15位（即 `rd` 位置）。
    *   **结论**：如果不知道 `lw` 没有 `rd` 只有 `rt`，这道选择题就必错。

#### 考法二：流水线数据冒险 (Data Hazards) —— **必考重难点**
*   **题目样子**：
    ```assembly
    1. lw   $1, 0($2)   ; 把数据读进 $1
    2. add  $3, $1, $4  ; 用 $1 做加法
    ```
    问：这里需要暂停（Stall）几个周期？
*   **解题逻辑**：
    *   你得看懂第一行：`lw` 的**目的寄存器**是 `$1`（也就是 `rt`）。
    *   你得看懂第二行：`add` 的**源寄存器**是 `$1`（也就是 `rs`）。
    *   因为第一行在写 `$1`，第二行马上要读 `$1`，所以产生了“写后读”（RAW）冒险。
    *   **如果你看不懂简写，你就找不到这种依赖关系。**

#### 考法三：机器码翻译（汇编 <-> 二进制）
*   **题目样子**：将机器码 `0x8FA40000` 翻译成汇编指令。
*   **解题逻辑**：
    *   你需要把十六进制拆成二进制。
    *   前6位是 Opcode，查表发现是 `lw`。
    *   然后你得知道接下来的5位是 `rs`，再下来5位是 `rt`。
    *   如果搞反了 `rs` 和 `rt` 的顺序，翻译出来的代码就是错的。

### 总结：你需要记什么？

不用死记硬背每一个指令的二进制码，但必须记住**数据流向**：

1.  **运算指令 (add/sub)**：`rd` 是终点，`rs` 和 `rt` 是起点。
2.  **取数指令 (lw)**：`rt` 是终点（从内存拿回来），`rs` 是基址。
3.  **存数指令 (sw)**：**特别注意！** `rt` 是起点（数据源），`rs` 是基址。
    *   *sw 是唯一一个 rt 作为源数据的 I 型指令常见特例。*

**一句话攻略**：
看到题目里的 `lw rt, rs, imm`，脑子里要马上浮现出一张图：**`rs` 和 `imm` 走进 ALU 算地址，算完去内存找数据，找完数据塞进 `rt`。** 这就是这道题的全部考点基础。
