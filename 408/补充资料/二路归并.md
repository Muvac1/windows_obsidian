#二路归并  是 #归并排序 (Merge Sort) 算法中的核心操作。它的任务是将两个**已经有序**的子序列合并成一个单一的、完全有序的序列。

例如，有两个已排序的序列 A = [1, 5, 8] 和 B = [2, 4, 9]，二路归并操作会将它们合并成一个新的有序序列 C = [1, 2, 4, 5, 8, 9]。

这个操作的基本思想是：
1.  使用两个指针，分别指向两个序列的起始位置。
2.  比较两个指针指向的元素，将较小的那个元素放入结果序列中，并将该指针向后移动一位。
3.  重复步骤2，直到其中一个序列的所有元素都被处理完毕。
4.  将另一个序列中剩余的所有元素直接复制到结果序列的末尾。

由于这个过程只需要从头到尾遍历两个序列一次，所以它的时间复杂度是线性的。如果两个序列的长度分别是 $m$ 和 $k$，那么总元素个数 $n = m+k$，归并操作的时间复杂度就是 $O(m+k)$，即 $O(n)$。
