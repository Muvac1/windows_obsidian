#编码方式的优缺点比较 

| 编码方案       | 自同步能力   | 直流分量 | 波特率 vs. 比特率                   |
| :--------- | :------ | :--- | :---------------------------- |
| **NRZ**    | 差       | 有    | $BaudRate = BitRate$          |
| **NRZI**   | 稍好 (对1) | 有    | $BaudRate = BitRate$          |
| **曼彻斯特**   | 好       | 无    | $BaudRate = 2 \times BitRate$ |
| **差分曼彻斯特** | 好       | 无    | $BaudRate = 2 \times BitRate$ |
