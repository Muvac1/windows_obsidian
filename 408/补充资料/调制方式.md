若某信道的数据传输速率为 9600 bps，波特率为 2400 Baud，则每个码元必须有多少种离散状态？该 #调制方式 可能是什么？
*   **解析**：
    *   已知 $C = 9600$ bps，$B = 2400$ Baud。
    *   首先计算每个码元携带的比特数 $k = C / B = 9600 / 2400 = 4$ bit。
    *   然后根据 $k = \log_2 N$ 反推 $N$：
        $N = 2^k = 2^4 = 16$
    *   因此，每个码元必须有16种离散状态。
    *   这种 #调制方式 可能是 16-PSK 或更常见的 **16-QAM** (16-状态正交振幅调制)。
