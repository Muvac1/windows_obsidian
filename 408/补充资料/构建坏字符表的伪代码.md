#构建坏字符表的伪代码  

```c
// 假设字符集大小为 256
void generateBadChar(char *pattern, int m, int bc[]) {
    // 1. 初始化：将表中所有值设为 -1
    for (int i = 0; i < 256; ++i) {
        bc[i] = -1; 
    }
    // 2. 填充：记录模式串中每个字符最后出现的位置
    // 注意这里遍历了模式串，但改变的是表中的值，而不是表的长度
    for (int i = 0; i < m; ++i) {
        int ascii = (int)pattern[i];
        bc[ascii] = i;
    }
}
```
