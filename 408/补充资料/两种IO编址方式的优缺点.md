  | 特性 | 独立编址 (Isolated I/O) | 统一编址 (Memory-Mapped I/O) |
    | :--- | :--- | :--- |
    | **地址空间** | 内存与I/O地址空间分离 | 内存与I/O共享同一地址空间 |
    | **访问指令** | 专用I/O指令 (`IN`, `OUT`) | 通用访存指令 (`MOV`, `LDA`) |
    | **地址识别** | 由指令类型区分 | 由地址码本身区分 |
    | **优点** | 不占用内存地址空间，指令清晰 | 指令集简单，访问方式灵活 |
    | **缺点** | 指令类型多，功能受限 | 占用内存地址空间，译码复杂 |
